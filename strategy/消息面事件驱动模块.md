# 消息面事件驱动模块（共享·环境分修正项）

> 生成日期：2026-07-02
> 本模块是**全策略共用的消息面/事件驱动判断口径**。短线/中线/长线策略在做环境判断时，用本模块对 `strategy/大盘环境模块.md` 的环境分做**定性修正与风险否决**。
> 数据取数**优先走 iFinD MCP 工具**（`search_news` / `search_notice` / `get_edb_data`），MCP 不可用时才回退脚本。
>
> **⚠️ 重要声明：本模块为消息面判断辅助框架，不构成任何投资建议。消息面驱动存在时效性与证伪风险，所有信号为事后确认而非预测。请独立决策、盈亏自负。股市有风险，投资需谨慎。**

---

## 〇、定位：修正项，不是买入触发器

> **消息面只修正"环境分级 + 仓位 + 方向"，绝不作为唯一买入理由。** 买入触发永远回到个股的「形态 + 资金 + 位置」三共振。

- 本模块**不机械加进大盘环境模块的 0-100 分**。原因：消息是定性的，机械打分会失真；更重要的是，机械给消息加分会直接诱发《短线炒股策略》第九条明令避开的**"追消息"坑**。
- 它的作用有三种：**降级否决**（利空）、**方向加权**（超预期利好，仅作主线参考）、**见顶警示**（利好出尽）。

---

## 一、消息分级（影响力从大到小）

| 级别 | 类型 | 典型事件 | 对 A 股的作用范围 |
|---|---|---|---|
| **L1 宏观流动性** | 货币政策 | 美联储加息/降息、非农、国内降准降息、MLF/LPR | 系统性，影响全市场风险偏好与估值锚 |
| **L2 产业政策** | 国家/行业政策 | 以旧换新、AI+、低空经济、设备更新、行业整顿 | 板块级，直接催化对应赛道 |
| **L3 行业景气** | 供需/价格/订单 | 涨价函、龙头订单、产能出清、海外映射 | 板块级，趋势性但发酵较慢 |
| **L4 个股事件** | 公司层面 | 业绩预告、并购重组、减持、中标、诉讼 | 个股级，注意真伪与预期差 |

> 级别越高，对**环境分级**的修正权重越大；级别越低，越偏个股层面（归入排雷/体检环节处理）。

## 二、事件 → 板块映射表（速查）

### 外因（海外传导到 A 股）

| 消息 | 传导路径 | A 股利好方向 | A 股利空方向 |
|---|---|---|---|
| 美联储**加息** / 非农超预期强 | 美元↑ → 人民币承压 → 北向流出 → 风险偏好↓ | 有限（避险有时反被强美元压制） | 成长股、科技、高估值赛道 |
| 美联储**降息** / 非农弱 | 流动性宽松预期 → 风险偏好↑ | 成长、创新药、贵金属、港股科技 | — |
| 美股科技 / 费城半导体**大跌** | 情绪与估值映射 | — | A 股算力、半导体、CPO、光通信、存储 |
| 美股科技**大涨** / AI 资本开支超预期 | 情绪映射 + 产业链订单预期 | A 股算力、光模块、PCB、液冷 | — |
| **地缘冲突** / 油价大涨 | 避险 + 成本上升 | 军工、黄金、石油石化、油服 | 航空、高耗能制造 |
| 人民币**贬值** | 汇兑 + 资金流 | 出口链（家电/纺织/机械）、黄金 | 航空、进口依赖、外债高企业 |

### 内因（国内政策/景气）

| 消息 | A 股利好方向 | 备注 |
|---|---|---|
| 降准 / 降息 / 社融超预期 | 券商、银行、高估值成长 | L1 级，影响全局风险偏好 |
| 产业扶持政策落地 | 对应赛道（AI、低空、机器人、半导体设备等） | 看是否超预期、有无真实订单 |
| 大规模 IPO（如长鑫存储登陆科创板） | — | **利空**：分流存量科技资金，警惕流动性冲击 |
| 中报/年报披露窗口 | 业绩超预期个股 | 资金转向基本面验证，题材降温 |

> ⚠️ 映射表是**方向参考**，不是买入清单。同一消息在不同环境下强度不同，务必结合环境分级与预期差判断。

## 三、预期差三问（消息能不能用的过滤器）

对任何消息，动手前先问三句：

1. **超预期吗？** 市场已 price in 的消息无驱动力；只有**超预期 / 预期外**的消息才有真正的推动。
2. **已兑现吗？** 利好落地当天常见「**利好出尽是利空**」；利空充分预期后**钝化即机会**。
3. **时效多久？** 判断是**一日游**（脉冲，冲高即走）还是**趋势级**（可持续跟踪）。一日游消息只做情绪不做趋势。

> 三问任一为否定/不利，消息即降权或作废，不作为操作依据。

---

## 四、消息面纪律（守住不追热点的底线）

- **消息只用于调仓位、选方向，不作为唯一买入理由**；买入仍需个股三共振。
- **不追消息涨停板**：消息驱动的一字板/加速板是接盘区，不是入场点。
- **不信单一消息源**：重大消息需交叉验证（`search_news` + `search_notice` 双源）。
- **证伪即离场**：买入逻辑基于某消息，一旦消息被证伪或预期差消失，无条件走。
- **区分内外因**：不是所有海外消息都直接影响 A 股，先走传导路径判断，再定方向。

## 五、如何修正环境分级（接口约定，写给 Agent）

```
1. 先按《大盘环境模块》取四维数据、算出 0-100 环境分与初始分级。
2. 再用本模块做定性修正（不改分值，改分级/仓位/禁区）：

   IF 出现 L1/L2 级重大系统性利空（且未充分兑现）
   THEN 环境分级下调一档（如 🟡→🟠），并将受冲击板块打入当日禁区。

   IF 出现 L1/L2 级超预期利好（通过预期差三问）
   THEN 不上调环境分级，但将对应方向纳入主线加权参考（买入仍需三共振）。

   IF 利好已充分兑现（利好出尽）
   THEN 不加分，反而对相关板块启动见顶警示，收紧止盈。

3. 修正后的分级与禁区，交回对应策略（短/中/长）按各自门槛执行。
4. 消息数据优先走 iFinD MCP：search_news（资讯）/ search_notice（公告）/ get_edb_data（利率/社融等宏观）。
5. 周末/节假日消息面仅作方向储备，不触发实时操作。
```

## 七、实战案例（2026-07-02）

- **事件**：Meta 宣布跨界云基础设施、对外出售过剩 AI 算力 → 引发"算力过剩"预期（L1/L2 级外因利空，超预期、未兑现）。
- **传导**：隔夜美股 + 日韩芯片股大跌 → A 股算力/半导体/CPO/光通信/存储全线杀跌，创业板收跌 5.71%。
- **修正动作**：环境分级下调，算力/半导体链当日打入禁区（不接下跌的刀）。
- **同步方向**：避险资金流入黄金（招金黄金等涨停）→ 但黄金方向受制于**当晚美国非农**（预期差未定），故仅观察不盘前押注。
