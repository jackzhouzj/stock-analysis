# 短线严格筛选策略（五道闸门·一票否决·量化打分）

> 生成日期：2026-06-26
> 最近优化：2026-07-07（融入"起爆/启动"形态核对维度，强化技术指标共振确认）
> 配套文档：`短线炒股策略.md`（方法论）。本文件是其**可执行的严格筛选版**——把"凭盘感判断"升级为"逐条打勾、不达标即淘汰"。
> 数据来源：板块层用 `plate-rotation` skill（开盘啦/同花顺双源）；个股资金/财务/排雷用 **同花顺 iFinD MCP 工具**。
>
> **⚠️ 重要声明：本文不构成任何投资建议、不预测任何个股/板块涨跌。所有阈值为经验默认值（标注"可调"），脱离大盘环境谈胜率无意义。短线是负和博弈，长期看多数散户亏损。请用闲钱、严控仓位、机械执行纪律，盈亏自负。A股入市需谨慎。**

## 〇、如何使用本清单（漏斗逻辑）

```
全市场
  │  闸门0：环境闸（不过 → 当日空仓，后面全部不做）
  ▼
今日候选板块
  │  闸门1：板块闸（双源共振，淘汰伪主线/退潮板）
  ▼
主线板块内个股
  │  闸门4：排雷闸【先跑·一票否决】命中任意一条 → 直接出局
  ▼
干净标的
  │  闸门2：资金闸（连续净流入，淘汰情绪虚火）
  ▼
有真金标的
  │  闸门3：形态+位置闸（扳机 + 安全位置 + 技术指标共振确认）
  ▼
进入打分卡 → 达最低准入分 → 盈亏比校验 → 候选买入 → 机械买卖纪律
```

**核心原则**
1. **排雷闸（闸门4）最先跑**——避免亏大钱比赚钱重要，命中即淘汰，不浪费精力打分。
2. **逐层淘汰，不补救**——任意一层不达标直接出局，不"觉得它其实还行"。
3. **每条标准都绑定取数方式**——不写"看资金流"，写"用哪个工具、什么口径、什么阈值"。
4. **宁可错过，不可做错**——本清单偏保守，目标是过滤掉 90% 的票，只留极少数共振标的。
5. **形态是概率，不是必然**——完全符合形态的票，事后大涨的是少数，更多是失败或假突破。真正保护你的是【止损纪律】，而不是赌它一定成。

## 闸门 0 — 环境闸（先决条件，决定今天开不开张）

> 环境不对，技巧再好也亏。本闸不过 → **当日空仓**，闸门 1-4 全部跳过。
> 📌 **取数与分级口径统一引用 `strategy/大盘环境模块.md`**：先按该模块算「环境分」，短线主看**赚钱效应 + 主线清晰度**（情绪最敏感），下表是短线在该模块基础上的具体阈值。

| 检查项 | 取数方式 | 淘汰阈值（可调） | 通过条件 |
|---|---|---|---|
| 大盘量能 | iFinD `index_highfreq_quotes`（上证/创业板，成交额） | 两市成交额较近 5 日均量**萎缩 >20%** | 量能持平或放大 |
| 指数趋势 | iFinD `index_data`（沪深300/创业板指 收盘、均线） | 指数**跌破 20 日线且加速下行** | 站上或回踩不破 20 日线 |
| 情绪周期（赚钱效应） | iFinD `index_highfreq_quotes`（涨停/跌停/上涨家数）+ `search_stocks`（连板梯队、炸板）+ plate-rotation `today` | **退潮**：涨停家数骤减、跌停家数放大、上涨家数占比 **<40%**、炸板率 **>40%**、连板高度断层 | **进攻**：涨停家数维持/增加、跌停稀少、上涨家数占比 **>60%**、炸板率 **<20%**、有连板梯队 |
| 主线清晰度 | plate-rotation `today` 双源 | Top 板块双源**完全不重叠**、无持续主线 | 至少 1 条双源共振主线 |

**判定规则**
- 4 项中**命中 ≥2 项淘汰阈值 → 当日空仓**。
- 命中 1 项 → **降仓至半仓以下**，只做最强主线核心。
- 0 项命中 → 正常进攻，进入闸门 1。

> 📌 **消息面修正（引用 `strategy/消息面事件驱动模块.md`）**：算出上表结论后，再用消息面模块做定性修正——
> - 出现 L1/L2 级重大系统性利空（未充分兑现，如隔夜美股科技崩盘映射）→ **环境结论下调一档 + 受冲击板块打入当日禁区**（不接下跌的刀）；
> - 出现超预期利好（过预期差三问）→ 不放松环境门槛，仅将该方向纳入闸门 1 主线加权参考；
> - 利好已兑现（利好出尽）→ 对相关板块启动见顶警示，收紧止盈。
> 消息面只改分级/仓位/禁区，**不作为买入信号**，买入仍需闸门 1-4 + 三共振。

> 备忘：周末/节假日接口返回上一交易日数据，plate-rotation 会输出 `PR-EMPTY: 今天是周末`，此时不做实时判断。

> ✅ **数据落地说明（已补齐·2026-07-02 校正）**：情绪维度现可**全量取数**，不再需要人工输入——
> - **涨跌停/上涨家数**：`index_highfreq_quotes`（实时快照，仅交易日内有效）。
> - **连板梯队高度**：`search_stocks` 查「连续涨停天数≥2，按连板数排序」→ 统计各板高度（如 5 板 X 家、4 板 Y 家）。
> - **炸板率**：`search_stocks` 查「最高价触及涨停价但收盘价低于涨停价」得炸板家数 ÷（炸板家数 + 收盘涨停家数）。
> - ⚠️ 查询坑：`search_stocks` 不认「炸板/连板」黑话，须翻译成**价格/涨停天数条件**才能解析。
> - 附：连板梯队若集中在 ST/垃圾股 → 属"退潮期资金躲进 ST 博弈"，是弱势信号而非进攻信号。

## 闸门 1 — 板块闸（只在双源共振主线里找票）

> 用 `plate-rotation` skill。只有**持续性 + 当日爆发同时具备**的板块才进入下一层。

### 取数命令
```bash
# {SKILL_DIR} = plate-rotation skill 实际安装目录，由 Agent 运行时自动解析
SKILL={SKILL_DIR}
PR=$SKILL/scripts/platerotat.py

python3 $PR today --source kaipan --n 10   # 强度分（持续性）
python3 $PR today --source ths    --n 10   # 涨幅%（当日爆发）
python3 $PR curve --source kaipan --days 20 # Top5 排名变化曲线
python3 $PR wangking <code> --days 20       # 候选板块妖王榜（龙头持续性）
```

### 硬性筛选标准（B-1/B-2 为硬门槛，B-3/B-4 为加权项）

| 编号 | 标准 | 阈值（可调） | 工具/口径 |
|---|---|---|---|
| B-1 | **双源共振** | 该板块在 kaipan 强度榜 **Top10** 且 ths 涨幅榜 **Top10** 同时出现 | `today` 双源比对 |
| B-2 | **持续性达标** | kaipan 近 **10 日上榜 ≥6 次**，且排名未出现断崖（无连续掉出 Top10） | `curve` 看断点 |
| B-3 | **龙头可接力** | 妖王榜存在 **上榜 ≥5 次** 的真核心龙头，且当日仍在领涨 | `wangking <code>` 看 count |
| B-4 | **产业链共振** | 主板块强势时，**上下游/细分概念至少 1 个同步上榜** | `today` 扫细分 |

> 判定规则：**B-1、B-2 为硬门槛必须通过**（保证是真主线）；B-3、B-4 作为打分卡加分项，不满足不直接淘汰——避免四条硬条件叠加后实际筛不出任何板块（宁可少一分，不空集）。

### 一票否决（板块层）
- ❌ 仅 kaipan 上榜次数高、**从未做出 #1**、长期 6-10 名晃 → **伪主线**，不参与。
- ❌ 之前长期未上榜、今天突然 #1 → **妖板**，二次确认前不在此板块选票。
- ❌ 主线退潮信号（强度排名连续 ≥3 日下滑）→ 该板块淘汰，转向资金流入的新方向。

> ⚠️ 口径铁律：**kaipan 强度分 与 ths 涨幅% 不可直接比较**（量纲不同），只能各自排序。**"强度/涨幅" ≠ 资金净流入**，情绪 ≠ 真金。

## 闸门 4 — 排雷闸（先跑·一票否决·命中即出局）

> 这是最能"避免亏大钱"的环节。**在花精力打分之前先跑此闸**，命中任意一条直接淘汰。

| 编号 | 雷点 | 取数工具（iFinD MCP） | 否决阈值（可调） |
|---|---|---|---|
| R-1 | 大股东/创始人**密集减持** | `get_stock_events`（减持事件） | 近 3 个月有**密集减持公告**，尤其"业绩好仍减持" |
| R-2 | **机构持仓骤降 / 户数激增** | `get_stock_shareholders` | 机构持股比例环比**大幅下降**或股东户数显著增加（筹码涣散） |
| R-3 | **监管处罚/信披违规/概念造假** | `search_notice` + `get_stock_events` | 存在处罚、问询、立案、蹭热点造假记录 |
| R-4 | 控股股东**高质押** | `get_stock_shareholders` / `get_stock_events` | 质押比例 **>40%** |
| R-5 | **财务虚胖** | `get_stock_financials` | 净利暴增但**营收下滑**；或经营现金流远低于净利；或应收/存货异常高企 |
| R-6 | **黑天鹅** | `search_notice` | 商誉大额减值、重大诉讼、业绩暴雷预告 |
| R-7 | **司法风险（深度验证）** | company-query-risk `get_legal_risk_count` → 异常时查 `search_dishonest_executed_person` / `search_illegality` | 失信被执行人>0 / 严重违法 / 破产重组 → **一票否决**；被执行/限高密集 → 重大警示 |

### R-5 财务虚胖的细化判定（重点）
- 用 `get_stock_financials` 同时取：净利润同比、**营收同比**、经营性现金流、应收账款、存货。
- **净利暴增先看营收增速是否同步**：只有净利暴增、营收停滞/下滑 → 低基数/扭亏陷阱 → 否决。
- 经营现金流 / 净利润 **< 0.5** 且持续 → 利润含金量存疑 → 否决。

### R-7 司法风险验证执行说明
1. 先通过 `get_stock_info` 获取**公司全称**。
2. 调用 company-query-risk `get_legal_risk_count`（公司全称）快速统计。
3. **一票否决**：失信被执行人>0 / 严重违法 / 破产重组 → 直接淘汰。
4. 被执行/限高密集 → 标注重大警示，在打分卡中额外扣分。
5. 大公司正常经营性诉讼（裁判文书数百条）不视为雷，关键看**失信/限高/行政处罚/股权冻结**。

> 判定规则：**命中任意 1 条 → 直接淘汰，不进入闸门 2/3。** 排雷宁可错杀。

## 闸门 2 — 资金闸（连续净流入 + 量价健康，淘汰情绪虚火）

> 通过排雷的票，验证是否有**真金持续进场**。区分"连续净流入"和"脉冲对倒"，同时验证量价配合是否健康。

| 编号 | 标准 | 取数工具（iFinD MCP） | 阈值（可调） |
|---|---|---|---|
| M-1 | **主力/特大单连续净流入** | `get_stock_performance`（资金流，口径=**净主动买入额**）· 多窗口口径见 `strategy/主力资金流分析模块.md` | 短线窗口=**近5日（大方向）+ 今日/近3日（边际）**；近 **3 个交易日中 ≥2 日**净流入，且**非单日脉冲对倒** |
| M-2 | **量价配合健康** | `stock_highfreq_quotes` / `get_stock_performance`；量价四象限口径见 `strategy/主力资金流分析模块.md` 第五节 | 必须符合"**上涨放量、回调缩量**"的健康结构；**放天量滞涨（顶背离）= 否决** |
| M-3 | **龙虎榜/融资真金**（如可得） | `get_stock_events` / 龙虎榜数据 | 游资/机构席位**净买入**，而非净卖出对倒 |

### M-2 量价健康度判定（新增·核心补强）

> "量是灵魂"——没有成交量配合的突破，多为假突破。量价关系是形态闸的前置验证。

| 量价状态 | 特征 | 判定 |
|---|---|---|
| ✅ 量增价升 | 上涨日放量（量 > 5 日均量）| 量价同向·健康上涨，资金在进攻 |
| ✅ 量减价跌（回调缩量）| 回调日缩量（量 < 5 日均量）| 卖方惜售·洗盘而非出货 |
| ❌ 量增价跌 | 放量阴跌 | 资金出逃，**一票否决** |
| ⚠️ 量减价升 | 上涨但缩量 | 升势缺乏支撑，不追高，等放量确认 |
| ❌ 放天量滞涨 | 高位巨量但股价冲高回落/不涨 | 顶背离·出货信号，**一票否决** |

### 口径校验（必须执行，防被误导）
- ✅ 必须认准 **"净主动买入额 / 主力净流入"**，**不是**毛流入额、不是成交额、不是涨幅。
- ✅ "区间净流入"要用**明确日期区间**取数，不依赖"最近N日"这类模糊口径（部分接口解析不准）。
- ❌ 单日脉冲式巨额净流入但次日即净流出 = **对倒诱多**，淘汰。

> 判定规则：**M-1 + M-2 必须同时通过（硬条件）**——有资金但量价不健康=高位出货诱多，有量价但无资金=情绪虚火。M-3 作为加权项。任一硬条件不过 → 淘汰。

## 闸门 3 — 形态 + 位置闸（扳机 + 技术指标共振 + 安全位置）

> 通过资金闸的票，最后看形态触发、技术指标共振确认和位置安全。**形态是扳机，指标是确认，位置决定赔率。**
> 📌 **核心认知**：所谓"起爆点"几乎都是涨完之后才被追认的——事前无法确定起爆，高评分≠起爆前。量化高分的票往往已涨了一波，追高分常常=追高。形态只是概率工具，止损纪律才是真正的保护。

### 3.1 形态触发（至少命中 1 种）

| 编号 | 形态 | 可勾选特征 | 经验胜率* |
|---|---|---|---|
| F-1 | **放量突破平台/新高** | 横盘 ≥N 日后**带量**突破前高/箱体（量 > 近期均量 **1.5 倍**）；突破K线为中阳/大阳实体 | 约 55-65% |
| F-2 | **强势股缩量回踩均线（龙回头）** | 主升后回调至 5/10 日线**缩量企稳**（回调量 < 上涨量 50%）；回调幅度 <20% | 约 55-65% |
| F-3 | **缩量整理末端突放量（均线粘合发散）** | 长期缩量后多均线粘合（5/10/20/60 日线间距收窄），量能异动放大，均线向上发散 | 约 50-58% |
| F-4 | **板块龙头首板/二板（情绪强期）** | 热门板块**第一个**涨停/晋级，板块有明确辨识度 | 约 50-60% |

\* 经验性区间，**非严谨统计**，随大盘环境剧烈波动。脱离环境谈胜率无意义。

### 3.2 技术指标共振确认（新增·起爆形态核对维度）

> 形态触发后，用技术指标做**二次确认**。指标不满足不直接一票否决，但会在打分卡中扣分（降低信心）。**避免在指标超买/严重乖离时追高。**

| 编号 | 指标 | 利好启动信号 | 警惕信号（扣分/不追） | 取数工具 |
|---|---|---|---|---|
| T-1 | **MACD** | 在 **0 轴附近金叉**，红柱开始放大（0轴上方更可靠） | 高位死叉、绿柱扩大 | `get_stock_performance` / `stock_highfreq_quotes` |
| T-2 | **KDJ** | **低位金叉**向上（J<20 区域金叉最佳） | J>100 超买区追入风险高 | `get_stock_performance` |
| T-3 | **BOLL（布林带）** | 开口**放大**，股价沿上轨或中轨上方运行 | 缩口或股价触下轨破位 | `get_stock_performance` |
| T-4 | **均线多头排列** | 5/10/20/60 日线由粘合转为**向上发散**、多头排列形成 | 空头排列、均线下压 | `get_stock_performance` |
| T-5 | **BIAS（乖离率）** | 短期正乖离在合理范围内（BIAS5 **< +8%**） | BIAS5 正乖离 **>10%**（远离均线，回踩概率大，不追） | `get_stock_performance` |
| T-6 | **WR（威廉指标）** | WR 从超卖区（>80）向上穿越 | WR 在极度超买区（<10）且持续 | `get_stock_performance` |

**判定规则**
- T-1~T-4 中**至少 3 项满足**利好信号 → 技术指标共振确认通过（打分卡满分）。
- 2 项满足 → 部分确认（打分中档）。
- <2 项或命中 T-5/T-6 警惕信号 → 打分低档；若 BIAS5>10% **且** J>100 同时命中 → 当日不追，等回踩。

> ⚠️ **指标是辅助确认，不是买入信号**：指标全绿仍需闸门 1-4 + 形态触发通过。指标是"给已触发形态加信心"，不是单独发现票的依据。突破初期 KDJ 可能较快进入超买区，需结合趋势判断：**强趋势中 KDJ 钝化是常态，弱势中才有意义**。


### 3.3 突破后确认（二次验证·提高胜率）

> **最佳买点往往不是突破当天，而是突破后的回踩确认。** 突破当日涨幅过大（远离均线）→ 不追，等回踩。

| 编号 | 确认标准 | 判定条件 | 操作含义 |
|---|---|---|---|
| C-1 | **回踩不破** | 突破后回踩**不跌破平台上沿/颈线/关键均线** | 支撑有效，二次确认成功 |
| C-2 | **回踩后再次放量上攻** | 回踩缩量后**再次放量阳线**（量>回踩期均量 1.3 倍）| 二次放量确认，胜率更高 |
| C-3 | **未快速跌回平台内** | 突破后 **3 个交易日内不跌回**平台/箱体内 | 有效突破 |

**判定规则**
- 突破当天可买入（F-1~F-4 触发），但：
  - 若突破当日涨幅 **>7%** 或 BIAS5>8%：**不追**，等回踩（C-1/C-2 满足后再入）。
  - 若突破后 **快速跌回平台内**（C-3 不满足）：判定为**假突破**，触发止损。
- 优先"突破 + 回踩确认（C-1+C-2）"的复合信号入场，而非追首根突破K线的高点。

### 3.4 位置安全（全部满足才通过，一票否决）

| 编号 | 标准 | 阈值（可调） | 取数工具 |
|---|---|---|---|
| P-1 | **不追加速连板** | 已 **≥3 连板**或当日加速妖股 → 否决（追高=赌接力） | `get_stock_performance` 技术形态 |
| P-2 | **距启动涨幅可控** | 平台突破型（F-1/F-3）：自启动累计涨幅 **<30%**；创阶段新高型：放宽至 **<50%**，但须量价健康、非加速末端 | `get_stock_performance` |
| P-3 | **优先低位共振** | "板块强 + 个股还没大涨 + 资金刚进"的潜伏位 | 综合 B/M/F |
| P-4 | **形态服从趋势** | 逆大势（大盘弱势/板块退潮）的个股突破 → 否决；只做**顺势突破** | 综合环境闸+板块闸 |

> 判定规则：形态命中 ≥1 种（3.1）**且** 位置安全 P-1~P-4 全部满足 → 通过，进入打分卡。

| 维度 | 满分 | 评分要点 |
|---|---|---|
| 板块强度（闸门1） | 20 | 双源共振 +8；持续性 ≥6 次 +6；龙头可接力 +3；产业链共振 +3 |

| 形态质量（闸门3.1） | 20 | F-1 放量突破 +20；F-2 龙回头 +18；F-3 均线粘合发散 +15；F-4 龙头首二板 +12 |
| 技术指标共振（闸门3.2·新增） | 10 | ≥3 项利好信号 +10；2 项 +6；<2 项 +2；命中超买警惕信号 +0 |
| 位置安全（闸门3.4） | 15 | 低位潜伏+回踩确认(C-1/C-2) +15；低位首次突破 +12；中位 +6；接近上限 +0 |
| 排雷边缘度（闸门4） | 10 | 治理指标离阈值越远越安全：质押 <10% + 近 3 月无减持 + 现金流/净利 >0.8 → +10；有单项接近阈值（如质押 30-40%、温和减持）→ +5；多项贴阈值 → 0（已命中阈值者在闸门4已出局，此列区分"干净"与"擦边"）|
| **合计** | **100** | |

**准入线（可调）**
- **总分 ≥80** → 一档候选（可纳入观察/小仓位）。
- **总分 65-79** → 二档（仅环境为强进攻期时考虑，半仓内）。
- **总分 <65** → 放弃。

## 盈亏比校验（打分达标后必过，不过则放弃）

> 短线赚钱的本质是"**亏小赚大**"。胜率 55–60% 必须配合 ≥2:1 的盈亏比，账户才是正期望。

### 计算方法
```
上行空间 = 入场价 → 最近压力位（前高 / 平台上沿 / 整数关 / 缺口）
下行空间 = 入场价 → 止损位（破支撑位 / 机械止损 -5%~-8%）
盈亏比 = 上行空间 ÷ 下行空间
```

### 铁律
```
IF 盈亏比 < 2:1
THEN 直接放弃，无论形态多漂亮、板块多强、打分多高
ELSE 进入买卖纪律执行
```

> 这也是"不追加速妖股"的数学解释：高位股上方没空间、下方是深坑，盈亏比天然差。

## 机械买卖纪律（if-then 规则，禁止临时改）

> 📌 仓位/熔断/集中度口径统一以 `strategy/资金与风险管理策略.md` 为准；卖出细则（止盈分批/减仓/换仓）见 `strategy/卖出决策策略.md`。本节为短线快速引用。

### 仓位
- **固定风险法（先算风险，再定仓位）**：`单票仓位金额 = (账户总额 × 单笔风险容忍度) ÷ 止损幅度`，单笔风险容忍度默认 **1%**（可调）。
  - 例：账户 100 万、止损幅度 -6% → 单票仓位 =（100万 × 1%）÷ 6% ≈ 16.7 万（约 16.7%）。
  - 含义：**止损越宽、仓位越小**，保证每笔的风险敞口恒定，而不是所有票都顶格买。
- **上限封顶**：一档 ≤20%，二档 ≤10%；固定风险法算出的仓位与上限**取小值**。
- **总仓位随环境闸调整**（命中1项降至半仓内）；退潮期总仓位 ≤50%；环境闸不过 → 0 仓。

### 买点
```
IF 形态触发当下（F-1~F-4 之一成立）AND 全部闸门通过 AND 打分达标 AND 盈亏比≥2:1
   AND (突破当日涨幅<7% OR 已等到回踩确认C-1/C-2)
THEN 买入
ELSE 不预判、不抄底下跌途中的票、不挂"以后可能会涨"的单
```

### 止损（无条件机械执行）
```
IF 跌破买入逻辑（破支撑 / 放量下杀 / 资金转净流出 / 板块退潮）
   OR 浮亏触及止损线
   OR 突破后快速跌回平台内（假突破）
THEN 无条件清仓，不补仓、不与趋势对抗
```
- **止损幅度按个股波动率自适应**（优先用 iFinD MCP `get_risk_indicators` 取波动率/beta）：
  - 低波动票（日均波动 <5% 或 beta<1）→ 收紧至 **-5%**；
  - 中波动票 → **-5%~-8%**（默认）；
  - 高波动题材票（日均波动 >8% 或 beta>1.5）→ 放宽至区间上沿或用 **1.5×ATR**，避免正常震荡被误扫。
- 单笔止损幅度**硬上限 -8%**，配合上方固定风险法控制敞口。

### A股制度性风险应对（涨跌停 + T+1，必须内化）

> 机械止损在 A股有两个制度性失效场景，不处理就是"纸面止损"。

- **涨跌停 / 一字板止损失效**：个股一字跌停时卖单无法成交，`-8%` 止损会被直接穿越，连续跌停可从 -8% 滑到 -30%+ 都卖不掉。应对：
  - 排雷闸对**易连续跌停品种**（ST/*ST、面值偏低、高质押暴雷、业绩地雷）直接规避，根本不参与；
  - 出现跌停苗头（尾盘跳水、放量破位）→ **不等触损线，尾盘主动撤**，不赌次日反弹；
  - 已一字跌停无法卖出 → 次日集合竞价**挂跌停价排队**优先成交，不恋战。
- **T+1 止损空窗**：当日买入次日才能卖，盘中止损有一天延迟，易被跳空低开穿越。应对：
  - **买入日尾盘决断**：若当日走势不及入场预期（未按形态放量拉升 / 资金未跟）→ 次日开盘无条件处理，不等触损；
  - 对隔夜跳空敏感的高位票，降低单票仓位或不做，规避跳空穿损。

### 止盈
```
IF 放天量滞涨 OR 跌破关键均线（如 5/10 日线）OR 主力资金转净流出（趋势性）
   OR MACD 高位死叉 + 量能萎缩
THEN 减仓 / 清仓（分批止盈见《卖出决策策略》）
ELSE 强势持有，看量价跟踪
```

## 账户级风控（单票纪律之上的熔断）

> 散户真正爆仓，**死于连亏后的报复性交易**，不是死于单笔。

### 三道熔断线（可调）

| 熔断类型 | 触发条件 | 执行动作 |
|----------|----------|----------|
| 单日止损 | 账户当日回撤达 **-3%** | 当日停手，不再开新仓 |
| 连亏熔断 | 连续亏损 **3 笔** | 进冷静期，降频/半仓/空仓观察 1–3 日 |
| 月度回撤 | 当月回撤达 **-10%** | 暂停短线，强制复盘找原因 |

### if-then 规则
```
IF 触发任意一道熔断
THEN 停止开新仓，执行对应动作
ELSE 正常执行单票纪律
```

> **情绪纪律**：不在大涨上头时追单（FOMO）；不在套牢后死抗、改口"做长线"自我安慰。

## 假突破识别与防范（新增·核心避坑清单）

> 形态触发后**失败是常态**。以下信号出现时，大概率是假突破，需立即止损或不入场。

| 信号 | 特征 | 识别取数 | 处置 |
|---|---|---|---|
| **无量突破** | 突破日成交量 < 5 日均量 | `stock_highfreq_quotes` | 不入场；若已入场次日无放量跟进 → 撤 |
| **放巨量长上影** | 突破K线巨量但收盘留长上影线（实体<1/3）| 日线 OHLC | 高概率诱多，不追；已入场 → 次日不涨即撤 |
| **快速跌回平台** | 突破后 1-3 日跌回原平台/箱体内 | `get_stock_performance` | 假突破坐实 → 无条件止损 |
| **资金当日净流出** | 突破日主力资金反而是**净流出** | `get_stock_performance`（净主动买入额）| 形态突破但资金未跟 → 信号矛盾，降仓或观望 |
| **逆大势孤军突破** | 大盘弱势/板块退潮时个股独涨 | 环境闸+板块闸 | P-4 已否决；漏过则视为高风险假突破 |

> **一句话**：**量是确认突破真伪的终极标尺**。没有量的突破 = 假突破预定。

## 必查口径备忘（防被工具误导）

1. **板块"强度分/涨幅" ≠ 元单位资金净流入**——口径不同，不可混用。
2. **行业级"流入额(合计)"常是毛流入/成交额**，不是净额——认准"**净主动买入额**"。
3. **"区间净流入"用明确日期区间取数**——"最近N日"在部分接口解析不准。
4. **净利同比"暴增"先看营收增速**——只有净利暴增多为低基数/扭亏陷阱。
5. **双源数值各自排序**——kaipan 强度分 与 ths 涨幅% 量纲不同，不能跨源比大小。
6. **`value=10.5`+`wu.png` = 当日未上榜**——不是排名 10.5 名，不参与平均。
7. **T+1 维度**——今天的龙头是昨天资金买入的，看资金切换信号窗口前移 1 天。
8. **指标钝化**——强趋势中 KDJ/WR 会长期超买钝化，此时指标失效，看均线趋势 + 量价为主。
9. **BIAS 乖离**——远离均线（BIAS5>10%）时追入，即使形态好也大概率回踩，等回踩不追高。

```
环境闸：命中≥2项淘汰 → 空仓

资金闸：净主动买入近3日≥2日净流入 + 非脉冲对倒 + 量价健康（上涨放量回调缩量）
形态闸：放量突破/龙回头/均线粘合发散/龙头首二板（任1）
技术共振：MACD 0轴金叉 + 均线多头发散 + BOLL开口 + KDJ低位金叉（≥3项确认）
位置闸：不追≥3连板 + 启动涨幅<30% + BIAS5<8%不追高 + 顺势不逆势

打分卡：≥80一档，65-79二档，<65放弃
盈亏比：≥2:1 才做
纪律：止损-5~-8%机械执行，放天量滞涨/破均线/资金流出/MACD死叉止盈
```

> **跟主线、先排雷、验真金、看量价、等扳机、查指标、控位置、守纪律**——比攻略更狠的地方在于：**每一关都设硬阈值，不达标直接淘汰，宁可错过 100 次，不做错 1 次。**

## 附录：起爆形态核对维度速查表（闸门3参考）

> 以下维度是闸门3（形态+指标+位置）的结构化核对项，**事前用于提高入场信心，事后用于复盘归因**。不要求每条全满足（那样几乎选不出票），但满足越多，成功概率越高。

### A. 趋势与结构
- [ ] 经历了较长时间的横盘整理/缩量筑底（时间越久，突破能量越足）
- [ ] 均线由多线粘合转为向上发散（5/10/20/60日）→ 对应 T-4
- [ ] 股价站上关键均线（尤其 MA60/MA120，由空头转多头）
- [ ] 突破前期重要高点 / 平台上沿 / 颈线 → 对应 F-1

### B. 量能（最关键）
- [ ] 突破时明显放量（量比 > 1.5，远超 5 日均量）→ 对应 F-1/M-2
- [ ] 上涨放量、回调缩量（健康的量价配合）→ 对应 M-2
- [ ] 不是"无量阴跌后的孤量拉升"（警惕诱多）→ 对应假突破识别

### C. 技术指标
- [ ] MACD 在 0 轴附近金叉，红柱放大（0轴上方更可靠）→ T-1
- [ ] KDJ 低位金叉向上 → T-2
- [ ] BOLL 开口放大，股价沿上轨/中轨上方运行 → T-3
- [ ] 均线多头排列逐步形成 → T-4
- [ ] BIAS5 正乖离未过大（<8%）→ T-5
- [ ] 避免在 KDJ-J>100 且 WR 超买时追高

### D. 突破后确认
- [ ] 突破后回踩不破平台上沿/颈线/关键均线 → C-1
- [ ] 回踩后再次放量上攻（二次确认）→ C-2
- [ ] 没有快速跌回平台内 → C-3

### E. 位置与催化
- [ ] 不追加速连板（≥3连板否决）→ P-1
- [ ] 距启动涨幅可控（<30%/50%）→ P-2
- [ ] 所属板块走强（强势板块中的强势股，而非孤军）→ P-4
- [ ] 有基本面/题材催化配合（业绩拐点、行业景气、政策）→ 加分项
