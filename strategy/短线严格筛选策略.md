## 〇、如何使用本清单（漏斗逻辑）

```
全市场
  │  闸门0：环境闸（不过 → 当日空仓，后面全部不做）
  ▼
今日候选板块
  │  闸门1：板块闸（双源共振 + 龙头存在，淘汰伪主线/退潮板/无龙板）
  ▼
主线板块内个股
  │  闸门2：龙头聚焦闸（识别龙头 → 聚焦龙头/最强小弟，淘汰纯跟风）
  ▼
龙头 / 准龙头 / 最强小弟
  │  闸门3：排雷闸【一票否决】命中任意一条 → 直接出局
  ▼
干净标的
  │  闸门4：资金闸（资金代理指标持续改善 + 量价健康）
  ▼
有资金与量价确认的标的
  │  闸门5：形态+位置闸（扳机 + 安全位置 + 技术复合确认）
  ▼
进入量化排序 → 达最低准入分 → 盈亏比校验 → 可成交性检查 → 候选交易 → 机械纪律
```

**核心原则**
1. **龙头聚焦（最高优先级·2026-07-14 新增）**——短线只做龙头或最强小弟，不做纯跟风。龙头有辨识度溢价、流动性优势和退潮容错率，跟风股在分歧时率先被闷杀。
2. **排雷闸（闸门3）优先执行**——候选龙头确认后立即排雷，命中即淘汰，不浪费精力计算资金、形态和评分。
3. **逐层淘汰，不补救**——任意一层不达标直接出局，不"觉得它其实还行"。
4. **每条标准都绑定取数方式**——不写"看资金流"，写"用哪个工具、什么口径、什么阈值"。
5. **宁可错过，不可做错**——本清单偏保守，目标是过滤掉 90% 的票，只留极少数龙头标的。
6. **形态是概率，不是必然**——完全符合形态的票，事后大涨的是少数，更多是失败或假突破。真正保护你的是【止损纪律】，而不是赌它一定成。

## 闸门 0 — 环境闸（先决条件，决定今天开不开张）

> 环境不对，技巧再好也亏。本闸不过 → **当日空仓**，后续所有闸门全部跳过。
> 📌 **唯一环境口径引用 `strategy/大盘环境模块.md`**：先按该模块计算0–100环境分。本节只定义短线覆盖规则，不再另算第二套“命中项数量”结论。

| 检查项 | 取数方式 | 统一口径 |
|---|---|---|
| 大盘量能 | iFinD `index_highfreq_quotes` + 历史指数数据 | 收盘比较沪深市场完整交易日总成交额；盘中只与近5个交易日**相同时刻累计成交额**比较，禁止拿盘中值对比历史全天值 |
| 指数趋势 | iFinD `index_data`（沪深300/创业板指收盘、均线） | 短线看20日线，中期风险参考60日线；均使用前复权且只使用信号时点已完成的数据 |
| 情绪周期 | iFinD `index_highfreq_quotes`（涨停/跌停/上涨家数）+ `search_stocks`（连板梯队、炸板） | 上涨占比的分母排除停牌；ST、新股无涨跌幅限制期、退市整理股单独统计，不混入普通连板梯队 |
| 主线清晰度 | plate-rotation `today` 双源 | 按《大盘环境模块》离散档位评分，不自行覆盖分值 |

**唯一判定规则**
- 环境分 **≥75**：正常进攻，进入闸门1。
- 环境分 **50–74**：防守模式，总仓位半仓内，只做最强主线核心。
- 环境分 **<50**：短线空仓。
- 覆盖规则：赚钱效应得分为0时，环境等级最高只能为“防守”；赚钱效应与主线清晰度同时为0时，强制短线空仓。
- 任一关键维度缺失时不得臆测补分；按可得维度保守降级，并在结果中列出缺失项。

> 📌 **消息面修正（引用 `strategy/消息面事件驱动模块.md`）**：算出上表结论后，再用消息面模块做定性修正——
> - 出现 L1/L2 级重大系统性利空（未充分兑现，如隔夜美股科技崩盘映射）→ **环境结论下调一档 + 受冲击板块打入当日禁区**（不接下跌的刀）；
> - 出现超预期利好（过预期差三问）→ 不放松环境门槛，仅将该方向纳入闸门 1 主线加权参考；
> - 利好已兑现（利好出尽）→ 对相关板块启动见顶警示，收紧止盈。
> 消息面只改分级/仓位/禁区，**不作为买入信号**，买入仍需通过后续全部硬闸、排序与盈亏比校验。

> 备忘：周末/节假日接口返回上一交易日数据，plate-rotation 会输出 `PR-EMPTY: 今天是周末`，此时不做实时判断。

> ✅ **数据落地说明（已补齐·2026-07-02 校正）**：情绪维度现可**全量取数**，不再需要人工输入——
> - **涨跌停/上涨家数**：`index_highfreq_quotes`（实时快照，仅交易日内有效）。
> - **连板梯队高度**：`search_stocks` 查「连续涨停天数≥2，按连板数排序」→ 统计各板高度（如 5 板 X 家、4 板 Y 家）。
> - **炸板率**：`search_stocks` 查「最高价触及涨停价但收盘价低于涨停价」得炸板家数 ÷（炸板家数 + 收盘涨停家数）。
> - ⚠️ 查询坑：`search_stocks` 不认「炸板/连板」黑话，须翻译成**价格/涨停天数条件**才能解析。
> - 附：连板梯队若集中在 ST/垃圾股 → 属"退潮期资金躲进 ST 博弈"，是弱势信号而非进攻信号。

## 闸门 1 — 板块闸（只在双源共振主线里找票）

> 用 `plate-rotation` skill。只有**持续性 + 当日爆发同时具备**的板块才进入下一层。

### 取数命令
```bash
# {SKILL_DIR} = plate-rotation skill 实际安装目录，由 Agent 运行时自动解析
SKILL={SKILL_DIR}
PR=$SKILL/scripts/platerotat.py

python3 $PR today --source kaipan --n 10   # 强度分（持续性）
python3 $PR today --source ths    --n 10   # 涨幅%（当日爆发）
python3 $PR curve --source kaipan --days 20 # Top5 排名变化曲线
python3 $PR wangking <code> --days 20       # 候选板块妖王榜（龙头持续性）
```

### 硬性筛选标准（启动模式与延续模式二选一，不得混用）

| 编号 | 标准 | 启动模式 A（新主线首板/二板） | 延续模式 B（成熟主线分歧/回踩） |
|---|---|---|---|
| B-1 | **双源共振** | kaipan、ths 同时进入Top10，且至少一源进入Top3 | kaipan、ths 同时进入Top10 |
| B-2 | **持续性** | 最近3日排名/强度持续改善；允许首次共振，不要求近10日上榜6次 | kaipan近10日上榜≥6次，且无连续2日掉出Top10 |
| B-3 | **龙头条件** | 候选股是板块首个有效涨停/突破者，处于首板或二板；尚未形成历史辨识度时只能标记“候选龙头” | 妖王榜存在上榜≥5次的核心龙头，且当日未进入退潮 |
| B-4 | **产业链共振** | 上下游/细分至少1个同步上榜，作为加分项 | 上下游/细分至少1个同步上榜，作为加分项 |

**模式准入规则**
- 模式A必须通过 B-1A、B-2A、B-3A；用于发现新主线，不得借用模式B的历史数据事后确认首板。
- 模式B必须通过 B-1B、B-2B、B-3B；用于成熟主线延续，不得再描述为“龙头首发”。
- B-4只加分，不补救任何硬门槛。
- 板块净资金数据如可得，必须与板块强度交叉验证；如不可得，至少要求板块核心股通过后续M-1、M-2，禁止把强度/涨幅直接称为资金流入。

### 一票否决（板块层）
- ❌ 仅 kaipan 上榜次数高、**从未做出 #1**、长期 6-10 名晃 → **伪主线**，不参与。
- ⚠️ 之前长期未上榜、今天突然 #1 → 只能进入**启动模式A候选池**，按候选龙头小仓验证；不得冒充满足持续性的延续模式B。
- ❌ 主线退潮信号（强度排名连续 ≥3 日下滑）→ 该板块淘汰，转向资金流入的新方向。
- ❌ **板块龙头已断板/炸板且无新龙接力** → 板块进入分歧期，跟风更危险，暂不参与。

> ⚠️ 口径铁律：**kaipan 强度分 与 ths 涨幅% 不可直接比较**（量纲不同），只能各自排序。**"强度/涨幅" ≠ 资金净流入**，情绪 ≠ 真金。

## 闸门 2 — 龙头聚焦闸（短线核心）

> **短线的本质是做龙头。** 板块涨跌看龙头、资金进退看龙头、情绪冷暖看龙头。做跟风股最大的风险是"龙头一分歧，跟风直接闷杀"。本闸在板块闸通过后、排雷闸之前执行，目标是在主线板块内**精确识别龙头**，并确定你要做的是龙头本身还是龙头的最强小弟。
>
> 📌 **核心认知**：龙头不是"涨最多的票"，而是"最先表态、最受资金认可、有辨识度溢价、退潮时最后倒"的票。龙头有市场情绪的定价权——它活着，板块就没完；它倒了，板块就结束。

### 一、龙头的五维定义（必须同时满足 ≥3 条才认定为龙头）

| 维度 | 龙头特征 | 跟风股特征 | 取数方式 |
|---|---|---|---|
| **① 时间优先（先手）** | 板块第一个涨停/第一个突破的票（首发效应） | 比龙头晚 1-N 天启动，跟随放量 | plate-rotation `wangking`（首次上榜日期）+ `get_stock_performance`（涨停板日期） |
| **② 空间优先（高度）** | 板块内连板高度最高 / 累计涨幅最大 | 涨幅 < 龙头的 50%，或只有 1-2 板 | `search_stocks`（连板天数排序）+ `get_stock_performance` |
| **③ 资金优先（真金）** | 主力净流入金额**板块内最大**或前三 | 净流入远小于龙头，或已转流出 | `get_stock_performance`（主力净流入，近5日排序） |
| **④ 辨识度（市场共识）** | 提到这个板块，市场第一时间想到这只票；龙虎榜有知名游资/机构参与 | 无辨识度，"换谁都行"的跟风品种 | `get_stock_events`（龙虎榜席位）+ 市场讨论度 |
| **⑤ 抗跌性（退潮不先倒）** | 板块分歧日，龙头仍封板/小跌，跟风闷杀 | 板块分歧日，率先跌停或大阴 | `get_stock_performance`（板块分歧日涨跌幅排序） |

**判定规则**：满足 ≥3 条 → 确认为龙头；满足 2 条 → "准龙头"（可参与但降仓位）；<2 条 → 跟风股（原则上不参与，除非做"最强补涨"逻辑）。

### 二、龙头生命周期（决定什么时机介入）

> 龙头有明确的生命周期。**不同阶段的操作策略截然不同**——启动期胜率最高，加速期赔率差，退潮期是陷阱。

| 阶段 | 特征 | 操作 | 风险 |
|---|---|---|---|
| **① 启动期（首板/二板）** | 板块刚进入启动模式A，候选龙头首次涨停/突破 | ⭐ **候选介入期**：只能依据当时可得信息识别，尚未通过后续表现确认时不得事后标记为“真龙头” | 可能是一日游，需小仓和次日验证 |
| **② 发酵期（3-4板/趋势确立）** | 板块持续强势，龙头晋级，跟风梯队形成 | ✅ **仅模式B可参与**：等待分歧转一致，不在一致加速时追入 | 追高成本抬高，需严格止损 |
| **③ 加速期（≥5板/连续一字）** | 龙头连续缩量或一字，市场高度一致 | ⚠️ **不追加速**：无正常成交机会时直接放弃 | 极高（封不住可能快速下杀） |
| **④ 分歧期（首次炸板/断板）** | 龙头首次炸板或断板，市场出现分歧 | 🎯 若次日按预定义条件修复，可进入模式B；连续走弱则退潮确认 | 失败概率高，仓位必须轻 |
| **⑤ 退潮期（高位放量滞涨/均线走坏）** | 龙头高位放量滞涨、跌破5日线、资金代理指标转弱 | ❌ **绝不参与** | 极高 |

### 三、龙头参与策略（三种模式·优先级排序）

> 以下胜率与赔率均须通过点时回测验证；在完成样本外验证前，只表示交易假设，不代表已证实优势。

| 模式 | 介入标的 | 介入时机 | 适用阶段 | 仓位上限 | 验证状态 |
|---|---|---|---|---|---|
| **A. 候选龙头首发** | 候选龙头 | 启动模式A：首板/二板按可得信息确认 | ①启动期 | ≤20% | 待样本外验证 |
| **B. 龙头分歧转一致** | 已确认龙头 | 延续模式B：炸板回封或断板次日修复 | ②发酵 / ④分歧 | ≤10% | 待样本外验证 |
| **C. 最强小弟** | 板块内相对强度第2-3名且有独立催化 | 龙头无法正常成交时，小弟自身首板/启动 | ②③期间 | ≤10% | 待样本外验证 |

**模式A详解**
- 首板条件：通过 B-1A/B-2A/B-3A + 板块内首个有效涨停 + 催化可核验 + 非尾盘瞬时封板。
- 二板条件：首板次日高开≥3%且量能相对近5日**相同时刻**放大，板块仍满足启动模式A或已切换到延续模式B，且不是一字板。
- 该模式理论上兼顾早期位置与辨识度，但一日游概率未知，必须独立统计成交率、胜率、盈亏比和最大回撤。

**模式B详解**
- 分歧日识别：龙头炸板或断板，且收盘涨跌幅 **> -5%**；收盘涨跌幅 **≤ -5%** 视为明显走弱，不参与修复博弈。
- 转一致信号：次日竞价高开后，开盘阶段价格和成交量按预定义条件修复；不得用当天收盘后的“回封成功”反推盘中买点。
- 模式B失败风险较高，只允许二档仓位，并单独统计，不与模式A合并计算胜率。

**模式C详解**
- 当龙头已加速/一字无法买入时，在板块内找"最强小弟"（辨识度仅次于龙头、涨幅第2-3名、有独立催化的票）
- **绝不做"纯跟风"** — 补涨龙必须有自身逻辑（如产业链上下游、技术独立、事件催化独立），而非"就因为同板块"
- **止损更严**：小弟在龙头分歧时**率先被闷杀**，止损必须提前到龙头首次炸板日

### 四、龙头资金特征（数据可得性分级）

> “主力净流入”是供应商基于成交方向和大单阈值推导的代理指标，不能证明真实交易主体，也不能单独识别对倒。必须与价格、成交量、龙虎榜和筹码变化交叉验证。

| 编号 | 核对项 | 取数 | 执行规则 |
|---|---|---|---|
| LM-1 | **板块内资金代理指标排名前三** | `get_stock_performance`（明确日期区间和指标单位） | 数据字段经样例验证后作为硬条件；字段缺失时标记不可评估，不得自动通过 |
| LM-2 | **封板质量** | Level-2/盘口队列（若可得） | 当前 `stock_highfreq_quotes` 未明确提供封单量，不设置“封单量/成交量>3”伪精确阈值；无盘口数据时不评分、不否决 |
| LM-3 | **资金代理指标先于涨幅改善** | 启动前1-3个已完成交易日 | 作为加分项，必须使用信号时点当时已经发布的数据 |

> 若后续接入Level-2，LM-2至少记录封单金额/自由流通市值、封单持续时间、开板次数和回封时间，再通过历史样本确定阈值。

### 五、龙头一票否决（特有雷点）

| 编号 | 否决条件 | 说明 |
|---|---|---|
| LR-1 | **已进入加速期（≥5板/一字）** | 追加速=赌接力，失败即闷杀（P-1 的龙头版升级） |
| LR-2 | **板块已有龙头断板且无新龙接力** | 老龙死、新龙未立=板块退潮，全板块不参与 |
| LR-3 | **辨识度不足/争议龙** | 市场对谁是龙头有争议（2-3只票争龙）→ 等确认再参与 |
| LR-4 | **龙头为ST/*ST/垃圾股** | ST龙头代表的是垃圾股博弈行情，非正常主线，不参与 |
| LR-5 | **龙头流通市值过小（<30亿）且已高位** | 小盘妖股流动性差，退潮时流动性枯竭无法止损 |

### 六、龙头与跟风的核心差异（决策思维转换）

> 做龙头和做跟风是两套完全不同的思维方式。**从今天起，默认只做龙头。**

| 维度 | 做龙头 | 做跟风（旧思维·弃用） |
|---|---|---|
| 选股逻辑 | "谁是板块灵魂？" | "哪个还没涨、便宜？" |
| 介入时机 | 辨识度确认即参与（右侧偏早） | 等龙头涨完了找补涨（右侧偏晚） |
| 止损依据 | 龙头逻辑被破坏（炸板不回封/断板走弱） | 自身形态破位 |
| 止盈依据 | 龙头生命周期结束（高位放量滞涨/均线走坏） | 被龙头连带闷杀被动出局 |
| 容错率 | 高（龙头有流动性，即使错了也能跑） | 低（跟风流动性差，一旦分歧直接跌停出不去） |
| 持仓心态 | "它是最强的，只要板块在它就在" | "龙头会不会带动我？求求别分歧" |

> **一句话**：做龙头是"站在市场情绪最确定的那一边"，做跟风是"赌龙头能照顾到自己"。两者的风险收益完全不对称。

## 闸门 3 — 排雷闸（一票否决·命中即出局）

> 这是最能"避免亏大钱"的环节。**在花精力打分之前先跑此闸**，命中任意一条直接淘汰。

| 编号 | 雷点 | 取数工具（iFinD MCP） | 初始量化口径（待回测校准） |
|---|---|---|---|
| R-1 | 大股东/创始人减持 | `get_stock_events` | 近90日减持公告≥2次，且计划上限合计≥总股本1%；只按信号时点已公告内容计算 |
| R-2 | 机构持仓下降/户数增加 | `get_stock_shareholders` | 最近一期机构持股比例环比下降≥20%（相对变化），或股东户数环比增加≥20%；报告未披露时标记数据滞后 |
| R-3 | 重大监管风险 | `search_notice` + `get_stock_events` | 立案调查、重大处罚、财务造假认定一票否决；普通问询函只警示，不因“存在问询”直接否决 |
| R-4 | 控股股东高质押 | `get_stock_shareholders` / `get_stock_events` | 质押比例>40% |
| R-5 | 财务质量异常 | `get_stock_financials` | 净利润为正时，经营现金流/净利润连续2个报告期<0.5；或营收同比<0且净利润异常增长；应收/存货需与自身历史及行业分位比较 |
| R-6 | 重大负面事件 | `search_notice` | 重大诉讼、商誉减值、业绩预告下修等必须同时记录事件日期、金额占净资产比例和是否已被价格反映，达到预设严重度才否决 |
| R-7 | 司法风险 | company-query-risk相关工具 | 失信被执行人>0、严重违法、破产重组一票否决；被执行/限高密集需按近12个月次数和金额量化 |

### 排雷数据规则
- 只使用信号时间之前已经公开的信息；财报、股东户数和机构持仓不得回填未来披露数据。
- 公司简称必须先映射统一社会信用主体/上市公司全称，避免同名企业误匹配。
- 任一硬风险字段查询失败时，标记“排雷未完成”，候选股只进入观察池，不得按“无风险”通过。
- 上述初始阈值是待验证参数，不代表天然有效；季度校准必须保留样本外检验。

### R-7 司法风险验证执行说明
1. 先通过 `get_stock_info` 获取**公司全称**。
2. 调用 company-query-risk `get_legal_risk_count`（公司全称）快速统计。
3. **一票否决**：失信被执行人>0 / 严重违法 / 破产重组 → 直接淘汰。
4. 被执行/限高密集 → 标注重大警示，在打分卡中额外扣分。
5. 大公司正常经营性诉讼（裁判文书数百条）不视为雷，关键看**失信/限高/行政处罚/股权冻结**。

> 判定规则：**命中任意1条硬否决项 → 直接淘汰，不进入闸门4/5。** 排雷宁可错杀；非硬否决警示项进入评分和仓位约束。

## 闸门 4 — 资金闸（资金代理指标 + 量价健康）

> 通过排雷的票，验证资金代理指标与量价是否一致。**主力/特大单净主动买入额是供应商推导指标，不等于账户级真实资金流，也不能单独证明“对倒”。**

| 编号 | 标准 | 取数工具（iFinD MCP） | 阈值（可调） |
|---|---|---|---|
| M-1 | **资金代理指标持续改善** | `get_stock_performance`（先用样例响应核验字段、单位和方向） | 近5日累计净额为正，且近3个已完成交易日中≥2日为正；单日极值后次日转负只标记“脉冲风险”，不直接推断交易主体 |
| M-2 | **量价配合健康** | `stock_highfreq_quotes` / `get_stock_performance` | 最近5个已完成交易日总体呈上涨放量、回调缩量；高位放量长上影或放量跌破支撑否决 |
| M-3 | **龙虎榜/融资交叉验证**（如可得） | `get_stock_events` / 龙虎榜数据 | 仅作加分；席位净买入不等于后续必然上涨 |

**时点约束**：盘中下单只能使用T-1及更早的完整日线资金数据，加上截至下单时刻的盘中量价；若使用T日完整资金流和日线成交量，只能在T日收盘后生成信号并于T+1执行。

### M-2 量价健康度判定（新增·核心补强）

> "量是灵魂"——没有成交量配合的突破，多为假突破。量价关系是形态闸的前置验证。

| 量价状态 | 特征 | 判定 |
|---|---|---|
| ✅ 量增价升 | 上涨日放量（量 > 5 日均量）| 量价同向·健康上涨，资金在进攻 |
| ✅ 量减价跌（回调缩量）| 回调日缩量（量 < 5 日均量）| 卖方惜售·洗盘而非出货 |
| ❌ 量增价跌 | 放量阴跌 | 资金出逃，**一票否决** |
| ⚠️ 量减价升 | 上涨但缩量 | 升势缺乏支撑，不追高，等放量确认 |
| ❌ 放天量滞涨 | 高位巨量但股价冲高回落/不涨 | 顶背离·出货信号，**一票否决** |

### 口径校验（必须执行，防被误导）
- ✅ 必须认准 **"净主动买入额 / 主力净流入"**，**不是**毛流入额、不是成交额、不是涨幅。
- ✅ "区间净流入"要用**明确日期区间**取数，不依赖"最近N日"这类模糊口径（部分接口解析不准）。
- ⚠️ 单日脉冲式巨额净流入但次日即净流出，只能标记为**脉冲风险/持续性不足**；不得据此断言对倒，需结合龙虎榜、价格位置和筹码变化。

> 判定规则：**M-1 + M-2 必须同时通过（硬条件）**——有资金但量价不健康=高位出货诱多，有量价但无资金=情绪虚火。M-3 作为加权项。任一硬条件不过 → 淘汰。

## 闸门 5 — 形态 + 位置闸（扳机 + 技术复合确认 + 安全位置）

> 通过资金闸的票，最后看形态触发、技术指标共振确认和位置安全。**形态是扳机，指标是确认，位置决定赔率。**
> 📌 **核心认知**：所谓"起爆点"几乎都是涨完之后才被追认的——事前无法确定起爆，高评分≠起爆前。量化高分的票往往已涨了一波，追高分常常=追高。形态只是概率工具，止损纪律才是真正的保护。

### 3.1 形态触发（至少命中 1 种）

| 编号 | 形态 | 可执行定义 | 验证状态 |
|---|---|---|---|
| F-1 | **放量突破平台/新高** | 默认平台期≥10个交易日；收盘突破平台上沿，成交量>前5日均量1.5倍；N值作为待回测参数显式记录 | 待样本外验证 |
| F-2 | **强势股缩量回踩均线** | 回踩5/10日线，回调量<前一上涨波段均量50%，收盘未有效跌破支撑；回调幅度<20% | 待样本外验证 |
| F-3 | **缩量整理末端放量** | 5/10/20/60日均线最大偏离率进入预设阈值后重新向上，成交量显著高于前5日均量 | 待样本外验证 |
| F-4 | **板块候选龙头首板/二板** | 仅适用于启动模式A；必须是板块首个有效涨停/晋级且可正常成交 | 待样本外验证 |

> 上述形态不再预设55%–65%胜率。胜率、赔率和适用环境必须由含交易成本、T+1及不可成交约束的点时回测给出。

### 3.2 技术指标共振确认（新增·起爆形态核对维度）

> 形态触发后，用技术指标做**二次确认**。指标不满足不直接一票否决，但会在打分卡中扣分（降低信心）。**避免在指标超买/严重乖离时追高。**

| 编号 | 指标 | 利好启动信号 | 警惕信号（扣分/不追） | 取数工具 |
|---|---|---|---|---|
| T-1 | **MACD** | 在 **0 轴附近金叉**，红柱开始放大（0轴上方更可靠） | 高位死叉、绿柱扩大 | `get_stock_performance` / `stock_highfreq_quotes` |
| T-2 | **KDJ** | **低位金叉**向上（J<20 区域金叉最佳） | J>100 超买区追入风险高 | `get_stock_performance` |
| T-3 | **BOLL（布林带）** | 开口**放大**，股价沿上轨或中轨上方运行 | 缩口或股价触下轨破位 | `get_stock_performance` |
| T-4 | **均线多头排列** | 5/10/20/60 日线由粘合转为**向上发散**、多头排列形成 | 空头排列、均线下压 | `get_stock_performance` |
| T-5 | **BIAS（乖离率）** | 短期正乖离在合理范围内（BIAS5 **< +8%**） | BIAS5 正乖离 **>10%**（远离均线，回踩概率大，不追） | `get_stock_performance` |
| T-6 | **WR（威廉指标）** | WR 从超卖区（>80）向上穿越 | WR 在极度超买区（<10）且持续 | `get_stock_performance` |

**判定规则**
- T-1~T-4 中**至少 3 项满足**利好信号 → 技术指标共振确认通过（打分卡满分）。
- 2 项满足 → 部分确认（打分中档）。
- <2 项或命中 T-5/T-6 警惕信号 → 打分低档；若 BIAS5>10% **且** J>100 同时命中 → 当日不追，等回踩。

> ⚠️ **指标是同一价格/成交量序列的相关衍生量，不是多份独立证据**：MACD、KDJ、BOLL和均线同时转强只计为一个“技术复合确认”，不得按四个独立因子重复解释胜率。盘中交易使用T-1已完成指标；T日完整指标只能生成T+1信号。强趋势中的KDJ/WR钝化需结合均线和量价，不能机械反向。


### 3.3 突破后确认（二次验证·提高胜率）

> **最佳买点往往不是突破当天，而是突破后的回踩确认。** 突破当日涨幅过大（远离均线）→ 不追，等回踩。

| 编号 | 确认标准 | 判定条件 | 操作含义 |
|---|---|---|---|
| C-1 | **回踩不破** | 突破后回踩**不跌破平台上沿/颈线/关键均线** | 支撑有效，二次确认成功 |
| C-2 | **回踩后再次放量上攻** | 回踩缩量后**再次放量阳线**（量>回踩期均量 1.3 倍）| 二次放量确认，胜率更高 |
| C-3 | **未快速跌回平台内** | 突破后 **3 个交易日内不跌回**平台/箱体内 | 有效突破 |

**判定规则**
- 突破当天可买入（F-1~F-4 触发），但：
  - 若突破当日涨幅 **>7%** 或 BIAS5>8%：**不追**，等回踩（C-1/C-2 满足后再入）。
  - 若突破后 **快速跌回平台内**（C-3 不满足）：判定为**假突破**，触发止损。
- 优先"突破 + 回踩确认（C-1+C-2）"的复合信号入场，而非追首根突破K线的高点。

### 3.4 位置安全（全部满足才通过，一票否决）

| 编号 | 标准 | 阈值（可调） | 取数工具 |
|---|---|---|---|
| P-1 | **不追加速** | ≥5连板、连续一字或连续缩量加速否决；3-4板仅允许延续模式B且仓位≤10% | `get_stock_performance` 技术形态 |
| P-2 | **距启动涨幅可控** | F-1/F-3的启动点定义为首次放量突破平台日；F-4定义为首板前一日收盘。自启动累计涨幅默认<30%，创阶段新高型可放宽至<50%，均为待回测参数 | `get_stock_performance` |
| P-3 | **低位与赔率** | 不使用“还没涨”主观判断；必须计算到止损位和最近客观压力位的距离 | 综合 B/M/F |
| P-4 | **形态服从趋势** | 环境空仓或板块退潮时否决；防守环境只允许最强核心 | 综合环境闸+板块闸 |
| P-5 | **可成交性** | 一字涨停、流动性不足、预计冲击成本过高或无法定义合法委托价格时否决 | 实时行情/盘口 |

> 判定规则：形态命中≥1种且P-1～P-5全部满足才进入评分。创业板、科创板、北交所、ST、新股无涨跌幅限制期必须按当日实际涨跌停制度单独计算；不得用固定9.9%识别涨停。

## 量化排序卡（硬门槛通过后使用·满分100）

> 打分只用于**排序**，不能补救任何硬门槛，也不等价于上涨概率。相关指标不得重复计分；同一事件同时满足多个描述时，只归入最主要维度。

| 维度 | 满分 | 评分规则 |
|---|---:|---|
| **环境与板块** | 20 | 环境≥75得5、50–74得2；双源均Top5得5、仅Top10得3；所选模式持续性强于最低门槛得5；产业链共振得3；板块排名继续改善得2 |
| **龙头相对强度** | 20 | 五维定义每项4分；≥12分才可称确认龙头，8分为准龙头，<8分只能按最强小弟规则处理 |
| **资金与量价** | 20 | M-1持续性7；M-2量价健康7；LM-1板块排名2；LM-2封板质量2；LM-3或M-3交叉验证2。LM-2无Level-2数据时本维度可用满分为18，再同比折算至20 |
| **入场结构** | 15 | F-1～F-4中最高质量形态最多8；C-1/C-2确认最多4；T-1～T-6作为一个复合确认最多3 |
| **位置、赔率与执行** | 15 | 距启动涨幅和支撑结构最多6；预估盈亏比≥2得5；非一字、流动性和委托可成交性最多4 |
| **排雷与数据质量** | 10 | 治理/财务指标远离否决阈值最多5；点时数据完整、单位明确、无未来数据最多3；流动性与主体映射可靠最多2 |
| **合计** | **100** | 缺失非关键加分字段时按该维度可用满分同比折算；缺失硬门槛字段时直接进入观察池，不得评分入选 |

**准入线（仅为排序阈值，待回测）**
- 总分≥80：一档候选；不代表80%胜率。
- 总分65–79：二档，仅环境进攻期考虑。
- 总分<65：放弃。
- 必须同时满足全部硬闸和盈亏比≥2:1；任何一票否决项不能用高分抵消。

> 评分权重必须通过训练期确定、验证期筛选、样本外期检验。没有样本外结果前，分数只表示规则符合程度。

## 盈亏比校验（打分达标后必过，不过则放弃）

> 短线赚钱的本质是"**亏小赚大**"。胜率 55–60% 必须配合 ≥2:1 的盈亏比，账户才是正期望。

### 计算方法
```
上行空间 = 入场价 → 最近压力位（前高 / 平台上沿 / 整数关 / 缺口）
下行空间 = 入场价 → 止损位（破支撑位 / 机械止损 -5%~-8%）
盈亏比 = 上行空间 ÷ 下行空间
```

### 铁律
```
IF 盈亏比 < 2:1
THEN 直接放弃，无论形态多漂亮、板块多强、打分多高
ELSE 进入买卖纪律执行
```

> 这也是"不追加速妖股"的数学解释：高位股上方没空间、下方是深坑，盈亏比天然差。

## 机械买卖纪律（if-then 规则，禁止临时改）

> 📌 仓位/熔断/集中度口径统一以 `strategy/资金与风险管理策略.md` 为准；卖出细则（止盈分批/减仓/换仓）见 `strategy/卖出决策策略.md`。本节为短线快速引用。

### 仓位
- **固定风险法（先算风险，再定仓位）**：`单票仓位金额 = (账户总额 × 单笔风险容忍度) ÷ 风险计算幅度`，单笔风险容忍度默认1%（待回测）。
  - `风险计算幅度 = max(技术止损幅度, 1.5×ATR幅度, 历史隔夜不利跳空分位数)`；无法取得后两项时采用更保守仓位，不得假设必能在止损价成交。
  - 例：账户100万、风险计算幅度6% → 理论仓位约16.7万，再与模式上限取小值。
- **上限封顶**：一档≤20%，二档≤10%；单一行业/产业链跨全部持仓合计≤35%。
- **总仓位**：环境≥75仍遵守账户机动现金要求；50–74短线半仓内；<50短线0仓。

### 信号时间与执行价格（禁止未来函数）

必须在交易记录中选择且只选择一种模型：

1. **收盘模型**：T日15:00后使用完整日线、最终炸板率和资金代理指标生成信号；最早于T+1执行。回测使用T+1开盘价、预定义VWAP或下一可成交价，不得按T日突破价成交。
2. **盘中模型**：只使用T-1及更早的完整日线指标，加上截至下单时刻的实时价格、成交量和盘口；量能必须与近5个交易日相同时刻比较。不得引用T日收盘价、最终日线指标、最终资金流或最终炸板率。

一字涨停、停牌、委托未成交必须记为“无成交”，不得从样本中删除。关键数据缺失或时间戳晚于决策时点时，信号无效。

### 买点
```text
IF 所选信号模型在决策时点已完成
   AND 全部硬闸通过
   AND 排序分达标
   AND 计入滑点后的预估盈亏比≥2:1
   AND 标的可正常成交
THEN 按预定义委托规则执行
ELSE 不交易；不得用收盘后信息补写盘中买点
```

### 止损（无条件机械执行）
```
IF 跌破买入逻辑（破支撑 / 放量下杀 / 资金转净流出 / 板块退潮）
   OR 浮亏触及止损线
   OR 突破后快速跌回平台内（假突破）
   OR 【龙头特有】龙头炸板不回封（当日炸板收跌且次日未反包）
   OR 【龙头特有】板块龙头断板且无新龙接力（板块退潮确认）
THEN 无条件清仓，不补仓、不与趋势对抗
```
- **止损幅度按个股波动率自适应**（优先用 iFinD MCP `get_risk_indicators` 取波动率/beta）：
  - 低波动票（日均波动 <5% 或 beta<1）→ 收紧至 **-5%**；
  - 中波动票 → **-5%~-8%**（默认）；
  - 高波动题材票（日均波动 >8% 或 beta>1.5）→ 放宽至区间上沿或用 **1.5×ATR**，避免正常震荡被误扫。
- 单笔止损幅度**硬上限 -8%**，配合上方固定风险法控制敞口。

### A股制度性风险应对（涨跌停 + T+1，必须内化）

> 机械止损在 A股有两个制度性失效场景，不处理就是"纸面止损"。

- **涨跌停 / 一字板止损失效**：个股一字跌停时卖单无法成交，`-8%` 止损会被直接穿越，连续跌停可从 -8% 滑到 -30%+ 都卖不掉。应对：
  - 排雷闸对**易连续跌停品种**（ST/*ST、面值偏低、高质押暴雷、业绩地雷）直接规避，根本不参与；
  - 对**T-1或更早已持有仓位**，出现尾盘跳水、放量破位时可主动撤退；当日新买仓位受T+1限制，不能写成“尾盘主动撤”。
  - 已一字跌停无法卖出 → 次日集合竞价**挂跌停价排队**优先成交，不恋战。
- **T+1 止损空窗**：当日买入次日才能卖，盘中止损有一天延迟，易被跳空低开穿越。应对：
  - **买入日尾盘决断**：若当日走势不及入场预期（未按形态放量拉升 / 资金未跟）→ 次日开盘无条件处理，不等触损；
  - 对隔夜跳空敏感的高位票，降低单票仓位或不做，规避跳空穿损。

### 止盈
```
IF 放天量滞涨 OR 跌破关键均线（如 5/10 日线）OR 主力资金转净流出（趋势性）
   OR MACD 高位死叉 + 量能萎缩
   OR 【龙头特有】龙头生命周期进入退潮期（高位放量滞涨 + 均线走坏 + 板块强度排名连降）
   OR 【龙头特有】板块出现新龙头且资金从老龙转向新龙（龙头更替=旧龙结束）
THEN 减仓 / 清仓（分批止盈见《卖出决策策略》）
ELSE 强势持有，看量价跟踪
```

### 龙头特有纪律补充（新增·2026-07-14）
```
【龙头首板/二板入场】
IF 首板当日封板结构不稳（反复开板、尾盘勉强封）
THEN 不参与，等次日确认
IF 二板开盘即一字（无买入机会）
THEN 不排队追一字，放弃或等分歧日

【龙头分歧转一致（模式B）入场】
IF 炸板/断板日收盘涨跌幅<=-5% OR 断板后连续2日走弱
THEN 放弃，不博弈明显走弱的分歧
IF 次日竞价高开>3% AND 开盘后量能相对历史相同时刻放大 AND 价格维持关键支撑
THEN 仅进入可交易状态，按盘中模型预定义委托执行；不得用收盘回封结果反推买点

【最强小弟（模式C）入场】
IF 龙头首次出现分歧（炸板/断板）
THEN 小弟必须当日减仓或次日开盘即走（小弟在龙头分歧时最先被闷杀）
```

## 账户级风控（单票纪律之上的熔断）

> 散户真正爆仓，**死于连亏后的报复性交易**，不是死于单笔。

### 三道熔断线（可调）

| 熔断类型 | 触发条件 | 执行动作 |
|----------|----------|----------|
| 单日止损 | 账户当日回撤达 **-3%** | 当日停手，不再开新仓 |
| 连亏熔断 | 连续亏损 **3 笔** | 进冷静期，降频/半仓/空仓观察 1–3 日 |
| 月度回撤 | 当月回撤达 **-10%** | 暂停短线，强制复盘找原因 |

### if-then 规则
```
IF 触发任意一道熔断
THEN 停止开新仓，执行对应动作
ELSE 正常执行单票纪律
```

> **情绪纪律**：不在大涨上头时追单（FOMO）；不在套牢后死抗、改口"做长线"自我安慰。

## 假突破识别与防范（新增·核心避坑清单）

> 形态触发后**失败是常态**。以下信号出现时，大概率是假突破，需立即止损或不入场。

| 信号 | 特征 | 识别取数 | 处置 |
|---|---|---|---|
| **无量突破** | 突破日成交量 < 5 日均量 | `stock_highfreq_quotes` | 不入场；若已入场次日无放量跟进 → 撤 |
| **放巨量长上影** | 突破K线巨量但收盘留长上影线（实体<1/3）| 日线 OHLC | 高概率诱多，不追；已入场 → 次日不涨即撤 |
| **快速跌回平台** | 突破后 1-3 日跌回原平台/箱体内 | `get_stock_performance` | 假突破坐实 → 无条件止损 |
| **资金代理指标转负** | 突破日完整资金代理指标为负 | `get_stock_performance` | 若采用收盘模型，M-1不通过则T+1不交易；若采用盘中模型，不得提前引用该收盘后结果，只能次日处理 |
| **逆大势孤军突破** | 大盘弱势/板块退潮时个股独涨 | 环境闸+板块闸 | P-4 已否决；漏过则视为高风险假突破 |

> **一句话**：**量是确认突破真伪的终极标尺**。没有量的突破 = 假突破预定。

## 必查口径备忘（防被工具误导）

1. **板块"强度分/涨幅" ≠ 元单位资金净流入**——口径不同，不可混用。
2. **行业级"流入额(合计)"常是毛流入/成交额**，不是净额——认准"**净主动买入额**"。
3. **"区间净流入"用明确日期区间取数**——"最近N日"在部分接口解析不准。
4. **净利同比"暴增"先看营收增速**——只有净利暴增多为低基数/扭亏陷阱。
5. **双源数值各自排序**——kaipan 强度分 与 ths 涨幅% 量纲不同，不能跨源比大小。
6. **`value=10.5`+`wu.png` = 当日未上榜**——不是排名 10.5 名，不参与平均。
7. **T+1 维度**——今天的龙头是昨天资金买入的，看资金切换信号窗口前移 1 天。
8. **指标钝化**——强趋势中 KDJ/WR 会长期超买钝化，此时指标失效，看均线趋势 + 量价为主。
9. **BIAS 乖离**——远离均线（BIAS5>10%）时追入，即使形态好也大概率回踩，等回踩不追高。

## 一页速查（口袋版·2026-07-20修订）

```text
环境闸：唯一使用环境分；≥75进攻，50–74防守半仓内，<50短线空仓
板块闸：启动模式A与延续模式B二选一，禁止拿成熟板块历史条件反证首板
龙头闸：五维每项4分；≥12分确认龙头，8分准龙头，<8分仅按最强小弟处理
排雷闸：只用决策前已披露数据；硬字段缺失=观察池，不得默认无雷
资金闸：资金流是代理指标；近5日累计为正+近3日≥2日为正，并与量价交叉验证
形态闸：形态仅是交易假设；技术指标作为一个复合确认，不重复计分
位置闸：≥5板/连续一字/缩量加速不追；3-4板仅模式B；必须验证可成交性
时点闸：收盘数据只能生成T+1信号；盘中模型只用T-1完整数据+当时实时数据
评分卡：满分100，仅排序；≥80一档、65–79二档，均不代表对应胜率
盈亏比：计入滑点后≥2:1；一字板、未成交和止损穿透必须纳入回测
风控：单票按风险计算幅度定仓，行业链≤35%，环境<50短线0仓
验证：模式A/B/C分别做walk-forward和样本外检验，未验证前不声称经验胜率
```

> **聚焦龙头、先排雷、验真金、看量价、等扳机、查指标、控位置、守纪律**——短线的alpha来自做龙头，不来自做跟风股的"便宜"。做最强的那一个，用纪律保护自己。

### 季度校准（训练、验证、样本外三段式）

> 禁止用同一批交易同时调参并证明策略有效。每次校准都必须固定数据版本、规则版本和交易成本假设。

1. **训练期**：估计候选阈值和权重，不报告为最终绩效。
2. **验证期**：在未参与拟合的数据上选择模型，限制参数数量和反复试验次数。
3. **样本外期**：锁定规则后只检验，不再修改；失败也必须保留记录。
4. **滚动前推**：下一季度只使用当时已经可得的历史数据重新训练，禁止回填未来财报、历史ST状态或当前板块成分。

每个模式A/B/C分别统计：信号数、可成交数、成交率、胜率、平均盈利、平均亏损、单笔期望、Profit Factor、最大回撤、最大连亏、持有期、滑点敏感性和95%置信区间。回测必须纳入佣金、印花税、过户费、T+1、一字板排队失败、涨跌停无法退出及退市样本。

- **重估形态优先级**：只有样本外结果稳定且样本量足够时才调整F-1～F-4权重。
- **校准打分卡**：优先检查增量解释力，MACD/KDJ/BOLL/均线等高度相关变量不得重复加权。
- **校准硬阈值**：每次只调整少量预先声明参数，避免多重检验和过拟合。
- **比较基准**：至少与“板块Top3中首个涨停、次日开盘等权买入”等简单基准比较，确认复杂规则是否产生增量价值。
- **留痕**：写入版本历史，记录数据区间、参数、修改原因、样本内/外结果和失败实验。

## 附录：起爆形态核对维度速查表（闸门5参考）

> 以下维度是闸门5（形态+指标+位置）的结构化核对项，事前用于形成可验证交易假设，事后用于复盘归因。满足项多不代表必然上涨，且相关指标不得重复计分。

### A. 趋势与结构
- [ ] 经历了较长时间的横盘整理/缩量筑底（时间越久，突破能量越足）
- [ ] 均线由多线粘合转为向上发散（5/10/20/60日）→ 对应 T-4
- [ ] 股价站上关键均线（尤其 MA60/MA120，由空头转多头）
- [ ] 突破前期重要高点 / 平台上沿 / 颈线 → 对应 F-1

### B. 量能（最关键）
- [ ] 突破时明显放量（量比 > 1.5，远超 5 日均量）→ 对应 F-1/M-2
- [ ] 上涨放量、回调缩量（健康的量价配合）→ 对应 M-2
- [ ] 不是"无量阴跌后的孤量拉升"（警惕诱多）→ 对应假突破识别

### C. 技术指标
- [ ] MACD 在 0 轴附近金叉，红柱放大（0轴上方更可靠）→ T-1
- [ ] KDJ 低位金叉向上 → T-2
- [ ] BOLL 开口放大，股价沿上轨/中轨上方运行 → T-3
- [ ] 均线多头排列逐步形成 → T-4
- [ ] BIAS5 正乖离未过大（<8%）→ T-5
- [ ] 避免在 KDJ-J>100 且 WR 超买时追高

### D. 突破后确认
- [ ] 突破后回踩不破平台上沿/颈线/关键均线 → C-1
- [ ] 回踩后再次放量上攻（二次确认）→ C-2
- [ ] 没有快速跌回平台内 → C-3

### E. 位置与催化
- [ ] 不追加速连板（≥3连板否决）→ P-1
- [ ] 距启动涨幅可控（<30%/50%）→ P-2
- [ ] 所属板块走强（强势板块中的强势股，而非孤军）→ P-4
- [ ] 有基本面/题材催化配合（业绩拐点、行业景气、政策）→ 加分项
