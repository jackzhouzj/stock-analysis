# 长线严格筛选策略（六道闸门·一票否决·量化打分）

> 生成日期：2026-06-26
> 配套文档：`长线投资策略.md`（方法论）。本文件是其**可执行的严格筛选版**——把“五维打分”升级为“逐条打勾、不达标即淘汰”。
> 数据来源：个股基本面/估值/治理用 **同花顺 iFinD MCP 工具**；行业景气用 `get_edb_data` / `sector_data`。
>
> **⚠️ 重要声明：本文不构成任何投资建议、不预测涨跌、不推荐任何个股/行业。所有阈值为经验默认值（标注"可调"），历史规律不代表未来。优质公司亦可能因行业变迁或黑天鹅长期下跌。请独立研究、独立决策、严控风险、盈亏自负。股市有风险，投资需谨慎。**

## 〇、如何使用本清单（漏斗逻辑）

```
全市场
  │  闸门0：赛道闸（行业不达标 → 不在此行业选票）
  ▼
高景气行业内公司
  │  闸门1：排雷闸【先跑·一票否决】治理/财务雷命中 → 直接出局
  ▼
干净公司
  │  闸门2：盈利质量闸（ROE/毛利/现金流不达标 → 淘汰）
  ▼
高质量公司
  │  闸门3：成长性闸（营收利润双高增且同步，剔除假成长）
  ▼
真成长公司
  │  闸门4：护城河闸（壁垒不清晰/在变窄 → 淘汰）
  ▼
有护城河的好公司
  │  闸门5：估值闸（贵就是风险，PEG/分位不达标 → 进观察池不买）
  ▼
好公司 + 好价格 → 打分卡 → 分批建仓 → 按基本面卖出
```

**核心原则**
1. **排雷闸（闸门1）最先跑**——长线持有时间长，治理雷暴露概率更大，命中即淘汰。
2. **盈利质量优先于成长速度**——先看"赚的是不是真钱"，再看"长得快不快"。
3. **好公司也要好价格**——估值闸不过的优质公司只进观察池，**不买贵**。
4. **每条标准绑定取数方式**——指定用哪个 iFinD 工具、什么口径、什么阈值。
5. **永久性本金损失才是风险**——本清单偏保守，宁可错过，不买错/不买贵。

## 闸门 0 — 赛道闸（行业决定天花板）

> 长牛多诞生于时代主导产业。行业不达标 → 不在此行业里选个股。

| 编号 | 检查项 | 取数工具（iFinD MCP） | 通过标准（可调） |
|---|---|---|---|
| S-1 | 行业空间大、低渗透率 | `get_edb_data`（行业产销/渗透率指标） | 处于成长期，渗透率有提升空间 |
| S-2 | 增速可持续 | `get_edb_data` / `sector_data` | 行业近 3 年复合增速正向且不靠单一年份 |
| S-3 | 格局向龙头集中 | `sector_data`（板块成分、集中度） | CR5 提升或龙头份额扩张 |
| S-4 | 非夕阳/非纯周期顶 | `get_edb_data` | 不属于长期产能过剩、被技术替代、强周期高位 |

**判定规则**
- S-1~S-4 **全部满足 → 通过**。
- 强周期行业仅在**周期底部**（景气指标处历史低分位）才考虑，否则淘汰。

## 闸门 1 — 排雷闸（先跑·一票否决·命中即出局）

> 诚信污点是长线最大杀手。**在花精力分析前先跑此闸**，命中任意一条直接淘汰。

| 编号 | 雷点 | 取数工具（iFinD MCP） | 否决阈值（可调） |
|---|---|---|---|
| R-1 | 实控人/大股东**持续减持** | `get_stock_events`（减持） | 持续/频繁减持，尤其"业绩好却猛减持" |
| R-2 | **财务造假/信披违规/处罚** | `search_notice` + `get_stock_events` | 存在处罚、立案、问询、造假记录 → **一票否决** |
| R-3 | 控股股东**高质押** | `get_stock_shareholders` / `get_stock_events` | 质押比例 **>40%**（爆仓风险传导） |
| R-4 | **商誉过高/并购买利润** | `get_stock_financials` | 商誉/净资产 **>30%**，或靠频繁并购堆利润 |
| R-5 | **主业不聚焦** | `get_stock_summary` / `get_stock_info` | 频繁跨界、关联交易复杂、主业模糊 |
| R-6 | **现金流与净利长期背离** | `get_stock_financials` | 经营现金流/净利润长期 **<0.8**（盈利质量存疑） |

> 判定规则：**命中任意 1 条 → 直接淘汰，不进入后续闸门。** 长线排雷宁可错杀。
> ESG 可作辅助：`get_esg_data` 评级偏低的，治理项重点复核。

## 闸门 2 — 盈利质量闸（先看是不是真钱）

> 通过排雷的公司，先验证**盈利质量**，再看成长。质量不达标，增长再快也淘汰。

| 编号 | 标准 | 取数工具（iFinD MCP） | 阈值（可调） |
|---|---|---|---|
| Q-1 | **ROE 长期高且稳定** | `get_stock_financials`（ROE，多期） | 近 **3-5 年 ROE >15%** 且稳定/提升 |
| Q-2 | **毛利率高且稳定** | `get_stock_financials`（毛利率，多期） | 毛利率处行业前列、无趋势性下滑（定价权投影） |
| Q-3 | **利润含金量** | `get_stock_financials`（经营现金流/净利润） | **>0.8**（利润是真金，非应收堆出） |
| Q-4 | **净利率稳定/提升** | `get_stock_financials` | 净利率无持续恶化 |

> 判定规则：Q-1、Q-3 为**硬条件**（必须通过）；Q-2、Q-4 为加权项。Q-1 或 Q-3 不过 → 淘汰。

---

## 闸门 3 — 成长性闸（剔除假成长）

> 质量达标后看成长。**营收与利润双高增且同步**，剔除低基数/非经常损益的假成长。

| 编号 | 标准 | 取数工具（iFinD MCP） | 阈值（可调） |
|---|---|---|---|
| G-1 | **营收持续增长** | `get_stock_financials`（营收同比，多期） | 近 3 年营收增速持续正向（非单年脉冲） |
| G-2 | **利润与营收同步** | `get_stock_financials` | 净利增速与营收增速匹配，**不只靠低基数** |
| G-3 | **增长可解释** | `get_stock_summary` / `search_news` | 来自需求扩张/份额提升/新品放量，非一次性收益 |
| G-4 | **非经常损益占比低** | `get_stock_financials` | 扣非净利润同步高增，非经常损益占比低 |

### 假成长识别（重点）
- **净利暴增先看营收增速**：净利暴增、营收停滞/下滑 → 低基数/扭亏陷阱 → 淘汰。
- **看扣非净利**：净利高增但扣非平平 → 靠投资收益/政府补贴/资产处置 → 淘汰。

> 判定规则：G-1、G-2 为硬条件；命中假成长信号 → 一票否决。

---

## 闸门 4 — 护城河闸（壁垒在变宽还是变窄）

> 最关键也最难量化。至少占一条壁垒**且可持续**，否则淘汰。

| 编号 | 壁垒类型 | 验证方式（iFinD + 定性） | 通过标准 |
|---|---|---|---|
| W-1 | 品牌/定价权 | `get_stock_financials`（毛利率持续高位） | 高毛利可持续 = 定价权投影 |
| W-2 | 技术/专利 | `get_stock_financials`（研发投入占比） + `search_news` | 研发投入持续，技术领先 |
| W-3 | 规模/成本优势 | `sector_data`（份额） | 龙头地位，成本曲线占优 |
| W-4 | 网络效应/转换成本 | 定性（`get_stock_summary`） | 用户黏性高、迁移成本高 |
| W-5 | **壁垒方向** | 综合 | 龙头份额 + 研发投入 → **壁垒在变宽而非变窄** |

> 判定规则：W-1~W-4 **至少满足 1 条**，且 W-5（壁垒变宽）成立 → 通过。讲不清"靠什么持续赚钱" → 淘汰（看不懂不买）。

## 闸门 5 — 估值闸（好公司也要好价格）

> 前 5 闸通过 = 好公司。本闸不过 → **只进观察池，不买**。再好的公司买在估值顶也可能多年不赚钱。

| 编号 | 标准 | 取数工具（iFinD MCP） | 阈值（可调） |
|---|---|---|---|
| V-1 | **PEG 合理** | `get_stock_financials`（PE / 净利增速） | **PEG < 1** 为理想成长区间 |
| V-2 | **PE/PB 历史低位** | `get_stock_financials`（PE/PB 分位数） | 处历史 **<50% 分位**，越低越好 |
| V-3 | **非价值陷阱** | 综合闸门1-4 | 低估值不是因基本面恶化/被减持压制 |

### 价值陷阱排查（重点）
- 低 PE 可能是**周期顶/衰退前兆**——结合赛道闸 S-4 复核。
- 便宜可能因**基本面在恶化**——若闸门2-3出现质量/成长下滑信号，低估值不构成安全垫。

> 判定规则：V-1 或 V-2 达标**且** V-3（非陷阱）成立 → 进入打分卡。否则进观察池等回调。

## 量化打分卡（最终准入）

| 维度 | 满分 | 评分要点 |
|---|---|---|
| 赛道质量（闸门0） | 15 | 高景气+格局集中 +15；一般 +8 |
| 盈利质量（闸门2） | 25 | ROE>15%稳定 +12；现金流/净利>0.8 +8；毛利率高位 +5 |
| 成长性（闸门3） | 20 | 营收利润双高增同步 +20；仅利润高增 +8 |
| 护城河（闸门4） | 25 | 壁垒清晰且变宽 +25；壁垒一般 +12 |
| 估值（闸门5） | 15 | PEG<1且低分位 +15；合理 +8；偏贵 +0 |
| **合计** | **100** | |

**准入线（可调）**
- **总分 ≥85** → 核心仓候选（可重点建仓）。
- **总分 70-84** → 卫星仓候选（小仓位/等更好价格）。
- **总分 <70** → 放弃或仅观察。

> 排雷闸命中者已在闸门1出局，不进入打分。打分是排序工具，达标 ≠ 立即买入，仍受估值闸约束。

## 机械建仓 / 持有 / 卖出纪律（if-then 规则）

### 建仓
```
分批建仓：不追高、不一次性满仓
IF 估值合理区（PEG<1 或 PE/PB 低分位）AND 全部闸门通过 AND 打分达标
THEN 分批买入（首仓试探，回调/逻辑验证后加仓）
越跌越买的唯一前提：基本面逻辑未变
```

### 仓位（可调）
- 单票上限：核心仓 ≤20%，卫星仓 ≤10%。
- 适度分散：**5-10 只、跨行业**，避免单一赛道/单票过度集中。
- 对宽基/优质核心资产可用**定投**平滑择时风险。

### 持有
```
拿得住的前提是看得懂
IF 基本面逻辑未变（成长/护城河/治理均健康）
THEN 持有，不因短期下跌恐慌、不因短期暴涨贪婪追高
```

### 卖出（基本面驱动，满足任一即卖，不看股价波动）
```
IF 基本面恶化（份额丢失/护城河变窄/行业转衰）
   OR 估值严重高估（PEG/分位到极端，透支多年增长）
   OR 出现治理雷（造假/实控人跑路式减持/重大违规）
   OR 发现更优标的（同等风险性价比更高，机会成本替换）
THEN 卖出
ELSE 持有（短期下跌若逻辑未变 = 加仓机会，不割肉）
```

## 持仓季度复检（每季财报后执行）

> 长线持有数月到数年，每季财报是核心验证节点。**财报披露后 1 周内必须复检**，确认买入逻辑是否继续兑现。

| 检查项 | 取数工具 | 继续持有 | 触发卖出 | 
|----------|----------|----------|----------|
| 营收增速 | `get_stock_financials` | 维持正向增长、与净利同步 | 营收连续两季下滑 |
| ROE / 毛利率 | `get_stock_financials` | 稳定或提升 | 趋势性下滑超 2 季 |
| 经营现金流/净利 | `get_stock_financials` | >0.8 | 连续 <0.5，利润含金量恶化 |
| 护城河指标 | `get_stock_financials` + `search_news` | 份额稳定/扩张、研发持续 | 份额丢失、技术被替代 |
| 估值分位 | `get_stock_financials` | 未到极端高位 | PEG>2 且分位>90%（严重透支） |
| 治理面 | `get_stock_events` / `search_notice` | 无新增减持/造假/违规 | 出现治理雷 |

### 复检规则
```
IF 全部检查项维持“继续持有”
THEN 持有不动，等待下一季复检
ELSE 命中“触发卖出”→ 执行卖出规则（基本面恶化/估值极端/治理雷）
```

> **注意**：短期股价下跌不触发复检卖出。长线只看基本面，不看股价波动——逻辑未变的下跌是加仓机会而非卖出理由。

## 必查口径备忘（防误判）

1. **净利同比"暴增"先看营收增速**——只有净利暴增多为低基数/扭亏假成长。
2. **看扣非净利**——剔除投资收益/补贴/资产处置等非经常损益。
3. **ROE 要看多期稳定性**——单年高 ROE 可能靠高杠杆或一次性。
4. **现金流/净利润 >0.8**——利润含金量的核心校验，长期背离即排雷。
5. **PE/PB 分位用历史区间取数**——低 PE 警惕周期顶/价值陷阱。
6. **质押比例 >40% 高度警惕**——爆仓风险会传导到股价。
7. **长短线不可混用**——长线票别短炒，短线套牢别改口"价值投资"。

## 一页速查（口袋版）

```
赛道闸：高景气+大空间+格局集中+非夕阳/非周期顶
排雷闸：减持/造假处罚/质押>40%/商誉>30%/不聚焦/现金流背离 → 命中即出局（先跑）
质量闸：ROE>15%稳定 + 经营现金流/净利>0.8（硬条件）
成长闸：营收利润双高增且同步 + 扣非同步（剔除假成长）
护城河：≥1条壁垒且在变宽，讲不清就不买
估值闸：PEG<1 或 PE/PB<50%分位 且非价值陷阱
打分卡：≥85核心仓，70-84卫星仓，<70放弃
纪律：分批建仓、5-10只分散、按基本面卖出、不追涨杀跌
```

> **选好赛道、先排干净雷、验真盈利、辨真成长、看清护城河、付好价格**——比攻略更狠的地方在于：**质量与排雷优先于成长速度，估值不过只观察不买，宁可错过，不买错、不买贵。**

## 长线 vs 短线（红线提醒）

| 维度 | 长线（本清单） | 短线（见《短线严格筛选策略》） |
|---|---|---|
| 收益来源 | 企业成长 + 估值修复 | 市场情绪 + 资金博弈 |
| 核心依据 | 盈利质量/护城河/估值 | 板块资金/形态/量价 |
| 持有周期 | 数月~数年 | 数日~数周 |
| 致命错误 | 用短线心态追涨杀跌 | 用长线心态扛套牢 |

> **最忌讳两套体系混用。** 这是亏损主因之一。
