# 中线严格筛选策略（七道闸门·一票否决·量化打分）

> 生成日期：2026-06-26
> 配套文档：`中线投资策略.md`（方法论）。本文件是其**可执行的严格筛选版**——把“趋势+景气+资金共振”升级为“逐条打勾、不达标即淘汰”。
> 数据来源：行业景气用 `get_edb_data` / `sector_data`；板块中期资金用 `plate-rotation` skill；个股趋势/业绩/资金/估值/治理用 **同花顺 iFinD MCP 工具**。
>
> **⚠️ 重要声明：本文不构成任何投资建议、不预测涨跌、不推荐任何个股/行业。所有阈值为经验默认值（标注"可调"），趋势与景气均可能反转。中线同时承担趋势破位与基本面证伪风险。请独立研究、独立决策、严控风险、盈亏自负。A股入市需谨慎。**

## 〇、如何使用本清单（漏斗逻辑）

```
全市场
  │  闸门0：周期/环境闸（中期趋势/风格不利 → 降仓或观望）
  ▼
顺风环境
  │  闸门1：行业景气闸（景气未向上/见顶 → 不在此行业选票）
  ▼
高景气行业内个股
  │  闸门2：排雷闸【先跑·一票否决】治理/业绩证伪雷命中 → 直接出局
  ▼
干净标的
  │  闸门3：趋势确立闸（趋势未走出/破位 → 淘汰，绝不左侧抄底）
  ▼
趋势向上标的
  │  闸门4：业绩/景气闸（业绩拐点未现/质量差 → 淘汰）
  ▼
基本面共振标的
  │  闸门5：资金闸（中期主力净流入不足 → 淘汰）
  ▼
有真金标的
  │  闸门6：估值闸（估值透支 → 进观察池不追）
  ▼
四共振标的 → 打分卡 → 趋势跟随建仓 → 移动止盈/破位止损
```

**核心原则**
1. **排雷闸（闸门2）最先跑**——中线持有数周到数月，治理与业绩证伪雷命中即淘汰。
2. **趋势确立是硬门槛（闸门3）**——中线只做右侧，**绝不左侧抄底下跌趋势**，趋势没走出来一律不碰。
3. **趋势 + 景气 + 资金 + 估值 四共振**才进打分卡，缺一不可。
4. **每条标准绑定取数方式**——指定用哪个工具、什么口径、什么阈值。
5. **破位即出局**——中线最大的纪律是不把波段票扛成套牢票。

## 闸门 0 — 周期/环境闸（中期趋势与风格）

> 中期趋势向下时中线胜率系统性下降。本闸不利 → 降仓 / 只做最强景气主线 / 观望。
> 📌 **大盘趋势/量能/情绪的取数与分级口径统一引用 `strategy/大盘环境模块.md`**：中线主看**指数趋势（60日线/周线）+ 流动性**，下表 C-1~C-4 是中线在该模块基础上叠加的风格与主线判断。

| 编号 | 检查项 | 取数工具 | 通过标准（可调） |
|---|---|---|---|
| C-1 | 指数中期趋势 | iFinD `index_data`（沪深300/创业板指，60日线/周线） | 指数站上 60 日线、周线多头 |
| C-2 | 市场风格 | `sector_data` / `index_data`（成长 vs 价值风格指数） | 站在当下占优风格的顺风口 |
| C-3 | 利率/流动性 | `get_edb_data`（10年期国债收益率、社融/M2） | 利率下行或平稳，流动性不收紧 |
| C-4 | 中期主线存在 | `plate-rotation` `today`/`curve`（持续性） | 至少 1 条中期资金持续流入的主线 |

**判定规则**
- 命中 ≥2 项不利 → **观望或仅极轻仓**。
- 命中 1 项 → 降总仓位，只做最强景气主线。
- 0 项 → 正常进攻，进入闸门 1。

> 📌 **消息面修正（引用 `strategy/消息面事件驱动模块.md`）**：中线对消息的敏感度低于短线，重点关注 **L1 流动性（利率/社融/降准降息）与 L2 产业政策** 对中期趋势与景气的影响——
> - L1/L2 级利空改变中期逻辑（如流动性收紧、行业政策打压）→ 降仓 / 暂停在受冲击行业新建仓；
> - 超预期产业政策强化景气 → 作为闸门 1 景气确认的加权佐证（仍需 `get_edb_data` 数据验证，不单凭消息）；
> - 一日游题材类消息不改变中线趋势判断，不追。

## 闸门 1 — 行业景气闸（中线的根基）

> 中线赚景气的钱。只在**景气向上 / 出现拐点**的行业里选票。

| 编号 | 标准 | 取数工具（iFinD MCP） | 阈值（可调） |
|---|---|---|---|
| I-1 | **景气趋势向上** | `get_edb_data`（行业产量/价格/库存/进出口/产能利用率） | 近 2–3 个季度景气指标持续改善，非单期脉冲 |
| I-2 | **景气拐点可验证** | `get_edb_data` + `search_news` | 需求扩张/价格回暖/订单回升有数据支撑 |
| I-3 | **板块中期资金** | `plate-rotation` `today`/`wangking`/`curve` | 板块中期资金持续流入、排名稳定上行 |
| I-4 | **非景气见顶** | `get_edb_data` | 景气未见顶回落、非产能过剩加剧、非政策打压 |

> 判定规则：I-1、I-4 为硬条件。景气见顶或证伪 → 该行业淘汰。
> ⚠️ 口径：**景气趋势 ≠ 当日板块涨幅**；行业"流入额(合计)"常是毛流入，认准净额。

## 闸门 2 — 排雷闸（先跑·一票否决·命中即出局）

> 在花精力分析前先跑此闸，命中任意一条直接淘汰。

| 编号 | 雷点 | 取数工具（iFinD MCP） | 否决阈值（可调） |
|---|---|---|---|
| R-1 | 实控人/大股东**持续减持** | `get_stock_events` | 趋势启动后高位持续减持 |
| R-2 | **业绩证伪风险** | `get_stock_financials` | 景气见顶、单季业绩环比转差、增长靠非经常损益 |
| R-3 | **造假/违规/处罚** | `search_notice` + `get_stock_events` | 存在处罚、立案、问询、造假 → 一票否决 |
| R-4 | 控股股东**高质押** | `get_stock_shareholders` | 质押比例 **>40%** |
| R-5 | **财务虚胖** | `get_stock_financials` | 净利暴增但营收下滑 / 应收存货高企 / 现金流远低于净利 |
| R-6 | **解禁/黑天鹅** | `search_notice` / `get_stock_events` | 限售解禁洪峰临近、商誉大额减值、重大诉讼 |

> 判定规则：**命中任意 1 条 → 直接淘汰**，不进入后续闸门。

## 闸门 3 — 趋势确立闸（中线硬门槛·只做右侧）

> 中线绝不左侧抄底。趋势没走出来一律淘汰。

| 编号 | 标准 | 取数工具（iFinD MCP） | 阈值（可调） |
|---|---|---|---|
| T-1 | **均线多头** | `get_stock_performance`（20/60 日线） | 20/60 日线向上发散，股价站稳其上 |
| T-2 | **趋势结构** | `stock_highfreq_quotes` / `get_stock_performance` | 周线突破下降趋势线/大平台，或日线高点抬高 |
| T-3 | **放量启动** | `get_stock_performance`（成交量） | 突破/启动时带量（量 > 近期均量 1.5 倍） |
| T-4 | **回踩不破**（加仓位） | `get_stock_performance` | 回踩 20/60 日线缩量企稳不破 |

> 判定规则：T-1 为硬条件；T-2/T-3 满足 ≥1；处下降趋势/均线空头 → **一票淘汰，不抄底**。

## 闸门 4 — 业绩/景气闸（基本面共振）

> 趋势确立后，验证基本面是否同步——区别于短线只看资金。

| 编号 | 标准 | 取数工具（iFinD MCP） | 阈值（可调） |
|---|---|---|---|
| F-1 | **业绩拐点/加速** | `get_stock_financials`（单季营收、扣非净利，多期） | 单季营收/扣非**环比或同比加速**，非低基数 |
| F-2 | **盈利质量** | `get_stock_financials`（ROE、毛利率、现金流/净利） | ROE 趋势向上、毛利率稳定、现金流/净利 >0.8 |
| F-3 | **增长可解释** | `get_stock_summary` / `search_news` | 景气/份额/新品驱动，非一次性收益 |
| F-4 | **公司处行业景气受益位** | `sector_data` / `get_stock_summary` | 是行业景气的核心受益标的，非边缘蹭概念 |

> 判定规则：F-1 为硬条件；F-1 证伪（业绩拐点是假的）→ 淘汰。

## 闸门 5 — 资金闸（中期主力净流入）

> 看周/月级别的持续净流入，不是单日脉冲。

| 编号 | 标准 | 取数工具（iFinD MCP） | 阈值（可调） |
|---|---|---|---|
| M-1 | **中期主力净流入** | `get_stock_performance`（资金流，口径=**净主动买入额**） | 近 **N 周/月** 主力持续净流入，非脉冲对倒 |
| M-2 | **机构动向** | `get_stock_shareholders` / `get_stock_events` | 北向/基金持仓提升、机构席位净买入 |
| M-3 | **量价配合** | `stock_highfreq_quotes` | 趋势上行放量、回调缩量；无放天量滞涨 |

> 判定规则：M-1 为硬条件（认准净主动买入额，用明确日期区间取数）。M-1 不过 → 淘汰。

## 闸门 6 — 估值闸（不追透支的趋势）

> 趋势+景气+资金都对，最后看估值是否已透支。透支 → 进观察池等回踩，不追高。

| 编号 | 标准 | 取数工具（iFinD MCP） | 阈值（可调） |
|---|---|---|---|
| V-1 | **估值分位不极端** | `get_stock_financials`（PE/PB 分位数） | PE/PB 未到历史 **>80% 极端高位** |
| V-2 | **PEG 合理** | `get_stock_financials`（PE / 增速） | 结合业绩增速，PEG 不显著 >1.5；G 用可持续预期增速，不用低基数单年高增速 |
| V-3 | **阶段涨幅可控** | `get_stock_performance` | 自趋势启动累计涨幅未过度透支（如 <50%，可调） |

> 判定规则：V-1 为硬条件；估值极端透支 → 进观察池，等回踩关键均线再评估，不追高。

## 量化打分卡（最终准入）

| 维度 | 满分 | 评分要点 |
|---|---|---|
| 行业景气（闸门1） | 20 | 景气向上+拐点验证 +20；一般向好 +10 |
| 趋势质量（闸门3） | 25 | 周线突破+均线多头+放量 +25；仅日线多头 +12 |
| 业绩/景气共振（闸门4） | 25 | 业绩拐点明确+核心受益 +25；拐点初现 +12 |
| 资金确认（闸门5） | 20 | 中期持续净流入+机构进 +20；脉冲 +5 |
| 估值安全（闸门6） | 10 | 分位中低+涨幅可控 +10；偏高 +3 |
| **合计** | **100** | |

**准入线（可调）**
- **总分 ≥80** → 一档候选（趋势确立可建仓）。
- **总分 65–79** → 二档（仅环境顺风时，轻仓试探/等回踩）。
- **总分 <65** → 放弃或观察。

> 排雷闸命中者已在闸门2出局，不进入打分。打分是排序工具，达标 ≠ 立即满仓，仍受趋势与估值约束。

## 赔率校验（打分达标后必过，不过则等更好买点）

> 中线同样是波段交易（破位即走、有明确止损与目标位），赔率不足的标的即使趋势/景气都对，也只是低性价比仓位。

### 计算方法
```
上行空间 = 入场价 → 中期目标位（前高 / 上一轮密集成交区 / 趋势通道上沿）
下行空间 = 入场价 → 止损位（破 60 日线 / 上升趋势线 / 机械止损 -8%~-10%）
赔率 = 上行空间 ÷ 下行空间
```

### 铁律（可调）
```
IF 赔率 < 1.5:1
THEN 不追高，进观察池，等回踩关键均线抬高赔率再入场
ELSE 进入建仓纪律
```

> 中线阈值（1.5:1）比短线（2:1）略低：中线靠趋势持续兑现，允许用时间换空间；但赔率仍是硬约束，不做"上方没空间、下方是深坑"的趋势末端接盘。

## 机械建仓 / 持有 / 卖出纪律（if-then 规则）

> 📌 仓位/熔断/集中度口径统一以 `strategy/资金与风险管理策略.md` 为准；卖出细则见 `strategy/卖出决策策略.md`。本节为中线快速引用。

### 建仓
```
分批建仓：趋势确立处试探，回踩关键均线不破时加仓
IF 全部闸门通过 AND 打分达标 AND 趋势确立（非下跌途中）
THEN 首仓试探 → 回踩20/60日线不破加仓
ELSE 不左侧抄底、不追已透支的趋势末端
```

### 仓位（可调）
- **固定风险法**：`单票仓位金额 = (账户总额 × 单笔风险容忍度) ÷ 止损幅度`，单笔风险容忍度默认 **1.5%**（中线止损更宽，容忍度可略高于短线）。算出值与下方上限**取小值**。
- 单票上限：一档 ≤25%，二档 ≤12%（中线可比短线集中，但仍设上限）。
- 总仓位随中期趋势与风格调整；闸门0不利 → 降仓/观望。

### 持有
```
IF 趋势未破（站稳60日线）AND 景气未证伪 AND 资金未持续撤出
THEN 趋势跟随持有，不被短期波动洗下车
```

### 卖出 / 移动止盈
```
止盈（趋势跟随）：
  跌破 20 日线 → 减仓
  跌破 60 日线 或 上升趋势线 → 清仓

无条件卖出（满足任一）：
  IF 趋势破位放量下杀
     OR 景气见顶/单季业绩环比明显转差（证伪）
     OR 主力/机构由净流入转持续净流出
     OR 估值分位到极端透支
     OR 出现治理雷（减持加速/违规/解禁砸盘）
  THEN 卖出

止损：建仓后趋势未走出且跌破买入逻辑 → 机械止损（-8%~-10%，可调，比短线略宽给趋势空间）
      止损幅度按个股波动率自适应（优先 iFinD MCP get_risk_indicators 取波动率/beta）：低波动票收紧、高波动票放宽，硬上限 -10%
      有效破位判定：以收盘价跌破 + 放量为准，或跌破幅度 >3%，避免盘中假破位与回踩加仓（T-4）在同一价位来回被打
```

## A股制度性风险应对

> 中线持有周期长，个股暴雷（连续跌停）的尾部风险高于短线，机械止损在跌停板下会失效。

- 排雷闸命中**高质押 / 高商誉 / 业绩证伪**的品种，是连续跌停高发区，宁可错过不参与。
- 遇一字 / 连续跌停无法止损 → 次日集合竞价挂跌停价排队出，不寄望反弹摊平。
- 重大利空停牌 → 复牌前评估补跌空间，必要时复牌首日无条件处理，不扛。

## 组合级系统性熔断（个股纪律之上）

> 个股「破位/证伪」卖出是第一层；当**系统性风险**来临（大盘中期趋势破位），需在组合层面统一降仓，而不是等每只票各自破位才反应。

| 触发条件 | 执行动作 |
|---|---|
| 大盘（沪深300/创业板指）**跌破 60 日线**且放量下行 | 中线总仓位降至 ≤50%，仅保留最强景气主线 |
| 大盘**周线破位**（跌破周线关键支撑/趋势线） | 中线总仓位降至 ≤30%，停止新开中线仓 |
| 环境分（见 `strategy/大盘环境模块.md`）**<30** | 中线清仓至底仓或空仓，等趋势重建 |

> 系统性降仓是"避免与大盘对抗"，与个股是否破位无关；大盘企稳、环境分回升后再按闸门重新建仓。

## 持仓定期复检（每周执行）

> 中线持有数周到数月，趋势和景气可能随时变化。**每周至少复检一次**，确认持仓逻辑未变。

| 检查项 | 取数工具 | 持有条件 | 卖出信号 |
|----------|----------|----------|----------|
| 趋势状态 | `get_stock_performance` | 站稳 60 日线、均线多头 | 跌破 60 日线放量下杀 |
| 景气验证 | `get_edb_data` / `search_news` | 行业景气指标继续向上 | 景气见顶回落、产能过剩 |
| 业绩跟踪 | `get_stock_financials` | 单季营收/扣非维持或加速 | 单季业绩环比明显转差 |
| 资金动向 | `get_stock_performance` | 主力中期持续净流入 | 转为持续净流出 |
| 治理面 | `get_stock_events` | 无新增减持/违规/解禁 | 出现新雷点 |

### 复检规则
```
IF 全部检查项维持“持有条件”
THEN 继续持有，趋势跟随
ELSE 命中任意“卖出信号”→ 执行卖出/移动止盈规则
```

## 必查口径备忘（防误判）

1. **中线看周/月级别趋势**——不被单日板块强度/涨幅带节奏。
2. **景气趋势 ≠ 当日涨幅**——用行业经济指标验证景气真伪。
3. **净利"暴增"先看营收和扣非**——区分真业绩拐点和低基数假象。
4. **资金认准"净主动买入额"**——毛流入/成交额不算，用明确日期区间取中期数据。
5. **PE/PB 分位结合利率环境**——利率上行期估值分位会系统性下移。
6. **趋势破位即走**——中线不把波段票扛成长线套牢票（头号亏损原因）。

## 一页速查（口袋版）

```
周期闸：指数站60日线+顺风格+流动性不收紧
景气闸：行业景气近2-3季持续向上+非见顶（硬条件）
排雷闸：减持/业绩证伪/违规/质押>40%/财务虚胖/解禁 → 命中即出局（先跑）
趋势闸：均线多头+周线突破+放量启动（只做右侧，不抄底）
业绩闸：单季营收/扣非加速+ROE向上+现金流/净利>0.8
资金闸：中期主力持续净流入（认净主动买入额）
估值闸：PE/PB分位<80% + 启动涨幅<50%
打分卡：≥80一档，65-79二档，<65放弃
纪律：破20日线减仓、破60日线清仓；止损-8~-10%
```

> **顺周期、选景气、只做右侧、验业绩、跟真金、控估值、破位走**——比攻略更狠的地方在于：**趋势确立是硬门槛绝不抄底，景气与业绩必须可验证，四维不共振直接淘汰，破位机械离场不扛单。**

## 中线 vs 短线 vs 长线（红线提醒）

| 维度 | 短线 | **中线（本清单）** | 长线 |
|---|---|---|---|
| 收益来源 | 情绪+资金博弈 | **中级趋势+景气兑现** | 企业成长+估值修复 |
| 核心依据 | 板块资金/形态/量价 | **趋势+景气+业绩+资金** | 盈利质量/护城河/估值 |
| 持有周期 | 数日~数周 | **数周~数月** | 数月~数年 |
| 致命错误 | 用长线心态扛套牢 | **波段票扛成长线套牢票** | 用短线心态追涨杀跌 |

> **最忌讳三套体系混用。** 中线破位不走、改口"长线持有"是头号亏损原因。
