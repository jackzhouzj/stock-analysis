## 〇、核心原则：卖出理由决定卖出方式

- **为什么卖，决定怎么卖**：止损（错了）要快要狠、止盈（对了）要慢要分批、换仓（有更好的）要果断。
- **卖出看的维度必须与买入对称**：短线因形态/资金/情绪买，就因形态破/资金撤/情绪退卖；长线因基本面买，就只因基本面卖，不看股价波动。
- **最忌讳**：亏损时找理由死扛（改口"做长线"）、盈利时拿不住（一涨就跑）。

---

## 一、卖出的三种性质（先归类，再决策）

| 性质 | 触发本质 | 卖出节奏 |
|---|---|---|
| **止损** | 判断错了（逻辑破坏） | **快、狠、全**——机械执行，不犹豫、不摊平 |
| **止盈** | 判断对了（兑现利润） | **慢、分批**——让利润奔跑，分批兑现 |
| **换仓** | 出现更好的标的 | **果断**——机会成本替换，同等风险选性价比更高的 |

---

## 二、止损决策（错了就走，第一优先级）

> 止损是保命，优先于一切。**触发即执行，不找理由、不摊平、不与趋势对抗。**

### 各线止损触发（阈值以对应严格筛选策略 + 《资金与风险管理策略》为准）
| 线 | 止损触发 |
|---|---|
| 短线 | 跌破买入逻辑（破支撑/放量下杀/资金转净流出/板块退潮）或触及 -5%~-8%（波动率自适应） |
| 中线 | 趋势破位（有效跌破 60 日线/趋势线，收盘+放量或跌幅>3%）或触及 -8%~-10% |
| 长线 | **不设价格止损**，只在基本面恶化/护城河变窄/治理雷时卖（见第三节） |

> 📌 **"资金转净流出"必须是趋势性流出，不是单日波动**：判定口径统一引用 `strategy/主力资金流分析模块.md` 的双窗口矩阵——短仓看近5日+今日、中仓看近20日+近5日，命中"持续派发"或"流入见顶转派发"格才触发资金面卖出，避免被单日脉冲洗下车。

### A股制度性风险（止损可能失效，必须内化）
- **一字/连续跌停无法成交**：出现跌停苗头→尾盘主动撤，不等触损；已一字跌停→次日集合竞价挂跌停价排队出。
- **T+1 空窗**：当日买入次日才能卖，买入日尾盘不及预期→次日开盘无条件处理。

## 三、止盈决策（对了慢慢卖，分批兑现）

> 止盈是本策略的核心难点。原则：**趋势没坏不全卖，用移动止盈让利润奔跑，用分批锁定落袋。**

### 3.1 移动止盈（趋势跟随，替代"涨多少卖"）
| 线 | 移动止盈规则 |
|---|---|
| 短线 | 放天量滞涨/跌破 5/10 日线/资金转净流出 → 减仓；破关键均线 → 清仓 |
| 中线 | 跌破 20 日线 → 减仓；跌破 60 日线/上升趋势线 → 清仓 |
| 长线 | 不看均线，只在估值极端高估（PEG/分位到极端）或基本面证伪时卖 |

> 📌 **量价卖出信号口径见 `strategy/主力资金流分析模块.md` 第五节**：**放天量滞涨（顶背离）、量增价跌（放量下杀）** 是两类核心减仓/清仓信号；**缩量创新高** 说明升势无量支撑，止盈可提前。量价定强弱、资金流定方向，两者共振才是可靠卖点。

### 3.2 分批止盈（锁定利润 + 保留弹性）
```
建议分批节奏（可调）：
- 到第一目标位（前高/压力位/赔率兑现2:1）→ 减1/3，锁定部分利润；剩余仓位仍占用资本并承担价格风险，继续执行止损
- 加速冲高 + 放天量滞涨 → 再减 1/3
- 趋势明确走坏（破关键均线/资金持续流出）→ 清掉剩余
```
> 分批的意义：**既不因贪婪坐过山车，也不因恐惧卖飞主升浪。**

### 3.3 止盈 vs 止损的不对称
- 止损要**快**（错了拖着只会更亏）；止盈要**慢**（对了拖着能赚更多）。
- 这就是"**亏小赚大**"盈亏比的执行落地：截断亏损、让利润奔跑。

## 四、基本面/治理驱动的卖出（中长线重点）

> 满足任一即卖，与股价涨跌无关：
- **基本面恶化**：份额丢失、护城河变窄、行业转衰、单季业绩环比明显转差（证伪）。
- **估值严重透支**：PEG/PE 分位到极端，透支多年增长。
- **治理雷**：造假、实控人跑路式减持、重大违规、高质押爆仓风险。
- **景气见顶**（中线）：行业景气指标见顶回落。

---

## 五、换仓决策（机会成本）

```
IF 发现更优标的（同等风险下性价比显著更高）
   AND 现有持仓的逻辑已弱化（不是最强逻辑）
THEN 换仓（卖弱买强），但不频繁换、不为换而换
ELSE 持有原仓
```
> 注意换仓的摩擦成本（印花税+佣金+滑点，单次往返约 0.1%-0.3%），差距不显著就不换。

---

## 六、卖出决策树（统一流程）

```
持仓复检时，按顺序判断：
1. 有没有触发止损/治理雷/基本面证伪？
   → 有 → 立即卖出（止损性质，快狠全）
2. 逻辑还在，但到了止盈信号（移动止盈线/目标位）？
   → 是 → 分批减仓（止盈性质，慢分批）
3. 逻辑还在，没到止盈信号？
   → 有更优标的 → 换仓；否则 → 继续持有
4. 全程受《资金与风险管理策略》的组合级/系统性熔断约束
```

## 七、卖出常见误区（血泪）

1. **止损犹豫**——"再等等会反弹"，扛成深套。止损必须机械。
2. **止盈太早**——一涨就跑，卖飞主升浪。用移动止盈替代"涨X%就卖"。
3. **亏损改口**——短线套牢改口"做长线"，是最典型的自欺。
4. **一次性清仓**——盈利时该分批兑现，不是全卖或全留的二选一。
5. **为卖而卖**——频繁止盈止损，摩擦成本吃掉利润。
6. **跌了不卖、涨了乱卖**——卖出理由应来自逻辑，不是来自股价情绪。

---

## 一句话总览

> **止损要快（错了就走）、止盈要慢（对了分批）、基本面坏了才卖长线、有更好的才换仓。** 卖出理由决定卖出方式，别用股价情绪代替逻辑判断。
