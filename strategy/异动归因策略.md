# 异动归因策略（盘中/盘后异动 · 快速定位原因 · 10 分钟出结论）

> 生成日期：2026-07-02
> 系统的使用场景里一直有"盘中异动快速原因分析"，本策略把它落地：个股或板块出现**涨停/跌停/放量拉升/闪崩**时，用固定流程快速定位原因，并联动到持仓（卖出决策）或候选（个股体检）。
> 取数**优先走 iFinD MCP**。
>
> **⚠️ 重要声明：本文不构成任何投资建议、不预测涨跌。异动归因是事后解释，不代表可预测后续走势。请独立决策、盈亏自负。股市有风险，投资需谨慎。**

---

## 〇、什么时候用

- 持仓股突然大涨/大跌/放巨量/闪崩：**先检查是否已触发预设止损、熔断或治理雷**。已触发时先提交风险降低型卖出，再并行归因；未触发时先归因再决定动作。
- 盘面某板块异动拉升 → 判断是真主线启动还是脉冲。
- 看到一只异动票想快速判断“值不值得跟” → 归因后转个股体检。

> **风险动作优先级**：归因不能成为延迟止损的理由。已触发规则时，风险动作优先；若因 T+1、停牌或跌停无法成交，记录未成交并继续占用风险额度，同时冻结该票加仓。

---

## 一、归因四步法（按顺序快速排查）

> 原则：**先查确定性信息（消息面），再查资金，再看联动，最后看位置。** 大多数异动能在前两步定位。

### 第 1 步：消息面（最优先，确定性最高）
| 查什么 | 工具（MCP 优先） | 判断 |
|---|---|---|
| 公司公告 | `search_notice` | 业绩预告、重组、中标、减持、处罚、澄清 |
| 突发新闻/舆情 | `search_news` | 行业政策、突发事件、传闻 |
| 重大事件 | `get_stock_events` | 解禁、减持、诉讼、股权变动 |

> 命中明确利好/利空消息 → 归因基本完成，直接进第 4 步定性。

### 第 2 步：资金面（无消息时的主因）
| 查什么 | 工具 | 判断 |
|---|---|---|
| 主力净流入 | `get_stock_performance`（净主动买入额） | 真金进/出，还是对倒 |
| 龙虎榜席位 | `get_stock_events` / 龙虎榜 | 游资/机构/北向谁在动 |
| 日内量价 | `stock_highfreq_quotes` | 拉升/跳水的量能、时点 |

### 第 3 步：板块联动（是不是被带动）
| 查什么 | 工具 | 判断 |
|---|---|---|
| 所属板块异动 | `plate-rotation` `today` / `sector_data` | 是板块整体异动还是个股独立行情 |
| 产业链共振 | `plate-rotation` | 上下游是否同步，判断主线真伪 |

### 第 4 步：技术位置（异动的位置含义）
| 查什么 | 工具 | 判断 |
|---|---|---|
| 位置/涨幅 | `get_stock_performance` | 低位启动 vs 高位加速（含义完全不同） |
| 均线/形态 | `get_stock_performance` | 突破/破位/见顶信号 |

## 二、异动类型 → 常见原因映射

| 异动 | 高频原因 | 重点排查 |
|---|---|---|
| 放量涨停 | 利好公告 / 板块主线 / 游资接力 | 消息面 + 板块 + 龙虎榜 |
| 一字涨停 | 重大利好 / 重组复牌 | 公告（第 1 步基本定位） |
| 放量跌停 | 利空 / 业绩暴雷 / 减持 / 监管 | 消息面 + 事件（排雷类） |
| 闪崩 | 大资金出逃 / 传闻 / 质押爆仓 | 资金面 + 质押/事件 |
| 高位放天量滞涨 | 主力派发出货 | 资金面（净流出）+ 位置 |
| 无消息放量拉升 | 资金抢筹 / 板块轮动 | 资金 + 板块 |

## 三、归因结论模板（10 分钟出）

```markdown
## 异动归因：{股票/板块}（{时间}）

- **异动表现**：{涨停/跌停/放量+X%/闪崩}，量能 {对比}
- **归因结论**：{一句话主因，如"重组复牌一字板" / "板块退潮资金出逃" / "无消息游资抢筹"}
- **证据链**：
  - 消息面：{有无公告/新闻，具体内容}
  - 资金面：{主力净流入/流出，龙虎榜席位}
  - 板块：{是否板块整体异动、产业链是否共振}
  - 位置：{低位启动 / 高位加速 / 破位}
- **可持续性判断**：{有基本面/资金支撑=可持续 / 纯情绪脉冲=不可持续}
- **风险提示**：{追高风险 / 消息证伪风险 / 出货风险}
```

## 四、归因后的动作联动

```
IF 异动票是我的持仓
THEN → 联动《卖出决策策略》：利空/出货信号 → 按止损/减仓规则处理；利好但高位放量滞涨 → 分批止盈

IF 异动票是候选/想跟
THEN → 联动《个股综合体检策略》：先排雷+体检，别只凭异动追进去

IF 是板块异动
THEN → 联动《短线严格筛选策略》闸门1：判断是否真主线，再决定是否在板块内选票
```

## 五、归因纪律（防被异动带节奏）

1. **规则触发时先控险，未触发时先归因**——预设止损、熔断或治理雷已触发时先执行风险降低动作，归因并行进行；未触发时不因异动盲目追涨杀跌。
2. **消息未证实按传闻处理**——传闻驱动的异动不可持续，不重仓参与。
3. **高位放量≠强势**——警惕主力借异动出货（放天量滞涨/冲高回落）。
4. **异动是事后解释，不是预测**——归因帮你理解发生了什么，不保证后续走势。
5. **一字板追不进就不追**——买不到的涨停不要用竞价高溢价去接。

## 一句话总览

> **查消息→验资金→看联动→定位置**，四步快速定位异动主因，再判断可持续性，最后联动卖出或体检——**先搞懂为什么动，再决定动不动。**
