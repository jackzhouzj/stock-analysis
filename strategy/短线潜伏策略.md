# 短线潜伏策略（左侧埋伏·吸筹先行·等点火）

> 生成日期：2026-07-03
> 配套文档：`短线炒股策略.md`（方法论）、`短线严格筛选策略.md`（右侧突破版）。
> **定位区别**：《短线严格筛选策略》是**右侧交易**——等放量突破「扳机」才买，买在启动确认后；本《短线潜伏策略》是**左侧偏埋伏**——在个股**尚未启动、低位缩量吸筹**阶段提前布局，赚「从潜伏到点火」的第一段拉升。两者互补，不是替代。
> 数据来源：板块/赛道层用 `plate-rotation` skill（开盘啦/同花顺双源）；个股资金/筹码/财务/排雷/催化用 **同花顺 iFinD MCP 工具**。
>
> **⚠️ 重要声明：本文不构成任何投资建议、不预测任何个股/板块涨跌。所有阈值为经验默认值（标注"可调"）。潜伏是"以时间换空间"的左侧思路，天然面临"潜伏失败/横盘破位/催化落空"风险，胜率不必然高于右侧突破。请用闲钱、严控仓位、机械执行纪律，盈亏自负。A股入市需谨慎。**

## 〇、潜伏策略的核心逻辑（为什么要潜伏）

```
右侧突破（严格筛选策略）：等放量破前高 → 确定性高，但已让出第一段涨幅，追高风险大
左侧潜伏（本策略）      ：在低位吸筹期提前埋伏 → 成本低、赔率高，但需承担"横久破位/催化落空"
```

**潜伏赚的是"三低一临近"的钱：**
1. **位置低**——远离压力位，跌无可跌，下行空间小 → 赔率天然好。
2. **情绪低**——市场没关注，无人抬轿，筹码还在低位。
3. **换手低**——缩量横盘，说明抛压枯竭、浮筹清洗完毕。
4. **催化临近**——有明确的、可预期的点火事件在不远的将来（业绩预告窗口、行业景气拐点、政策/订单预期、解禁减持利空已出清）。

> **一句话**：潜伏 = 在「主力悄悄吸筹 + 位置足够低 + 有明确催化预期」三者共振时，先于市场埋伏，等点火信号出现后加仓兑现，点火迟迟不来或破位则撤退。

**潜伏 ≠ 抄底套牢**：抄底是"下跌途中接刀、赌反转"；潜伏是"下跌已止、缩量筑底、吸筹迹象出现"后的埋伏。**没有止跌企稳 + 吸筹迹象，一律不潜伏。**

## 一、如何使用本清单（漏斗逻辑）

```
全市场
  │  闸门0：环境闸（系统性下跌不埋伏；但退潮/震荡期反而是潜伏好窗口）
  ▼
有催化预期的赛道
  │  闸门1：赛道催化闸（选"预期差 + 催化临近"的方向，不埋没故事的票）
  ▼
方向内个股
  │  闸门4：排雷闸【先跑·一票否决】命中任意一条 → 直接出局
  ▼
干净标的
  │  闸门2：吸筹闸（核心！主力吸筹迹象 + 筹码集中，淘汰无人问津的死票）
  ▼
有主力吸筹标的
  │  闸门3：位置形态闸（低位 + 止跌 + 缩量横盘 + 未启动）
  ▼
进入潜伏打分卡 → 达准入分 → 分批建底仓 → 等点火信号 → 点火加仓 / 破位或超时撤退
```

**核心原则**
1. **排雷闸（闸门4）最先跑**——潜伏周期比追涨长，暴露在治理雷下的时间更久，排雷更要严。
2. **没有吸筹迹象不埋伏**——潜伏的前提是"有人在悄悄买"，纯低位死票不是潜伏对象（会一直趴着）。
3. **催化必须可预期、有时间窗**——不埋"永远有故事、就是不涨"的票，必须有明确的、临近的点火由头。
4. **分批建仓 + 时间止损**——潜伏用轻底仓试错，点火才加仓；横盘超期 / 破平台一律撤，绝不死等。

## 闸门 0 — 环境闸（潜伏对环境的要求与追涨不同）

> 📌 **取数与分级口径统一引用 `strategy/大盘环境模块.md`**，先算「环境分」。
> **与短线追涨的关键差异**：追涨要求"进攻期 + 赚钱效应好"；潜伏**恰恰可以在震荡/退潮期布局**——市场冷清、筹码低位时正是埋伏良机。但**系统性下跌（不接刀）与个股所在赛道处于下行趋势**是潜伏的禁区。

| 检查项 | 取数方式 | 潜伏禁区（可调） | 潜伏适宜条件 |
|---|---|---|---|
| 大盘趋势 | iFinD `index_data`（沪深300/创业板指 收盘、均线） | 指数**跌破 60 日线且加速下行**（系统性下跌）→ 全面停手 | 指数在 20/60 日线附近震荡、缩量止跌 |
| 大盘量能 | iFinD `index_highfreq_quotes` | 恐慌性放量下杀途中 | **地量地价**（成交持续萎缩至阶段低位）→ 潜伏最佳窗口 |
| 情绪周期 | 见 `大盘环境模块.md` 取数配方 | 连续跳水、跌停潮 | **退潮末期 / 冰点期**（涨停少、无人参与）→ 埋伏低吸位 |
| 赛道趋势 | plate-rotation `curve` + `sector_data` | 目标赛道**强度排名持续下滑、资金持续流出** | 赛道在底部、资金流出放缓 / 有回流苗头 |

**判定规则**
- 大盘**系统性下跌（跌破 60 日线加速）** → 潜伏全面停手（左侧最怕系统性风险，会击穿所有平台）。
- 大盘震荡 / 退潮 / 冰点，但**未系统性破位** → **正常潜伏**（这是潜伏最好的环境，成本低、无人抢筹）。
- 大盘强势进攻期 → 潜伏窗口反而变窄（好票都启动了），此时以《短线严格筛选策略》右侧追涨为主，潜伏只做"主线里还没动的低位补涨候选"。

> 📌 **消息面修正（引用 `strategy/消息面事件驱动模块.md`）**：
> - 出现 L1/L2 级重大系统性利空且未兑现 → 暂停潜伏建仓，已建底仓按平台止损处理；
> - 目标赛道出现**超预期利好但尚未发酵**（过预期差三问）→ 这是潜伏的核心催化来源，纳入闸门1加权；
> - 目标个股的**利空（减持/解禁/问询）刚落地出清** → 属"利空出尽"，是潜伏的经典点火前置条件（见闸门4排雷需确认已出清）。

## 闸门 1 — 赛道催化闸（只埋伏"有预期差 + 催化临近"的方向）

> 潜伏不是随便找只低位票趴着，而是埋伏在**即将被点燃的方向**。没有催化预期的低位横盘 = 价值陷阱，会一直横下去。

### 取数命令 / 工具
```bash
# 板块层：找资金开始悄悄回流、但还没被市场充分定价的方向
SKILL={SKILL_DIR}/skills/plate-rotation-skill
python3 $SKILL/scripts/platerotat.py curve --source kaipan --days 20   # 看哪些板块从底部排名回升
python3 $SKILL/scripts/platerotat.py today --source kaipan --n 20      # 扫低位区将进未进的板块
```
> 个股/行业催化：`search_news`、`search_notice`、`get_edb_data`、`get_stock_events`。

### 硬性筛选标准（C-1 为硬门槛，C-2/C-3/C-4 为加权项）

| 编号 | 标准 | 阈值（可调） | 工具/口径 |
|---|---|---|---|
| C-1 | **有明确、临近的催化预期** | 存在**可预期的点火事件**且时间窗在**未来 1~8 周内**：业绩预告/财报窗口、行业景气拐点数据、政策/订单/新产品预期、大额利空出清 | `search_news` + `search_notice` + `get_edb_data` + `get_stock_events` |
| C-2 | **赛道位置低、资金回流** | 板块处于阶段底部区域，`curve` 显示强度排名**止跌回升**（非仍在下滑） | plate-rotation `curve` |
| C-3 | **预期差** | 市场预期低（无人关注、卖方覆盖少）但基本面/景气有边际改善 | `search_news`（关注度）+ `get_edb_data`（景气） |
| C-4 | **主线未启动的低位补涨位** | 若大盘有主线，目标是主线内"还没大涨"的细分/个股 | plate-rotation `today` 扫细分 |

> 判定规则：**C-1（催化临近）为硬门槛，必须通过**——没有点火由头的潜伏是价值陷阱。C-2/C-3/C-4 进打分卡加权。

### 一票否决（赛道层）
- ❌ **无任何可预期催化**、纯粹"跌得多所以想抄" → 价值陷阱，不埋。
- ❌ 赛道处于**明确下行趋势、资金持续净流出且无回流迹象** → 不接下跌赛道的刀。
- ❌ 催化是**"永远在路上"的模糊故事**（如"迟早会火"）、无明确时间窗 → 不埋。

## 闸门 4 — 排雷闸（先跑·一票否决·命中即出局）

> 潜伏持有周期更长，暴露在治理雷下的时间更久，**排雷标准比追涨更严**。命中任意一条直接淘汰。

| 编号 | 雷点 | 取数工具（iFinD MCP） | 否决阈值（可调） |
|---|---|---|---|
| R-1 | 大股东/创始人**减持进行中或将启动** | `get_stock_events`（减持事件） | 近 3 个月有减持公告、或**减持计划尚在实施窗口内**（潜伏期正好撞上减持=被压制） |
| R-2 | **筹码涣散**（与吸筹判断相反） | `get_stock_shareholders` | 股东户数持续激增、机构持股持续下降 → 无人吸筹，反在派发 |
| R-3 | **监管处罚/信披违规/概念造假** | `search_notice` + `get_stock_events` | 存在处罚、问询、立案、蹭热点造假记录 |
| R-4 | 控股股东**高质押** | `get_stock_shareholders` / `get_stock_events` | 质押比例 **>40%**（潜伏期长，爆仓/平仓风险更致命） |
| R-5 | **财务虚胖 / 持续恶化** | `get_stock_financials` | 净利暴增但营收下滑；经营现金流远低于净利；或基本面**持续恶化无拐点**（潜伏最怕基本面越埋越差） |
| R-6 | **黑天鹅 / 待兑现利空未出清** | `search_notice` + `get_stock_events` | 商誉大额减值、重大诉讼、业绩暴雷预告；**大额解禁/减持尚未落地**（利空未出尽不埋） |

### 潜伏专属排雷要点
- **"利空出清"是潜伏的前置，不是排雷豁免**：要确认减持/解禁/问询等利空**已落地兑现**，而非"还在头上悬着"。头上有刀的低位不是潜伏位，是绞肉机。
- **基本面必须无恶化趋势**：潜伏靠时间换空间，若基本面持续下滑，横盘平台会不断下移，止损被反复触发。

> 判定规则：**命中任意 1 条 → 直接淘汰，不进入闸门 2/3。** 排雷宁可错杀。

## 闸门 2 — 吸筹闸（核心！有主力在悄悄买才叫潜伏）

> 这是潜伏策略与短线追涨**最本质的差异闸**。追涨看"资金爆发式净流入 + 放量拉升"；潜伏看"**低位温和吸筹、缩量横盘中的资金迹象**"——主力在不拉高股价的前提下悄悄建仓。

| 编号 | 标准 | 取数工具（iFinD MCP） | 阈值/口径（可调） |
|---|---|---|---|
| A-1 | **筹码集中化**（吸筹的结果） | `get_stock_shareholders` | 股东户数**环比下降**（筹码向少数人集中）、或机构持股比例**环比上升** |
| A-2 | **多窗口主力资金趋势**（核心·见下） | `get_stock_performance`（资金流，口径=**主力/特大单净主动买入额**） | 用近 30 日 + 近 7 日双窗口判趋势，识别"持续流出=派发"还是"流出末端回流 / 温和吸筹"（判定矩阵见下） |
| A-3 | **缩量 + 大单托底** | `stock_highfreq_quotes` / `get_stock_performance`；量价口径见 `strategy/主力资金流分析模块.md` 第五节 | 下跌缩量（量减价跌·抛压枯竭）、回落有大单承接、地量地价；**警惕：量增价跌/放量下杀 = 出货，直接否决** |
| A-4 | **底部异动信号**（加分） | `get_stock_performance` 技术形态 / `stock_highfreq_quotes`；分时量型见 `主力资金流分析模块.md` 第七节 | 出现"缩量后某日温和放量小阳、次日不破"的试盘迹象；底部逐步抬高；**分时出现平头量(建仓量)/洗盘量** = 主力逢低吸筹的分时印证 |

### A-2 多窗口主力资金趋势分析（潜伏降风险的关键）

> 📌 **取数口径与判定矩阵统一引用 `strategy/主力资金流分析模块.md`**；本节为潜伏专属的窗口参数与用法。
> 潜伏最怕"埋在还在派发的票上"。单日资金流会被脉冲对倒骗到，必须看**趋势**：**近一月（30 日）看大方向、近 7 日看边际变化**，两窗口交叉判断。

**取数方式（必须用明确日期区间，不用"最近N日"模糊口径）**
```
用 get_stock_performance 取主力/特大单净主动买入额，分别拉两段明确区间：
  - 近 30 个自然日区间：{今日-30} ~ {今日}   → 求区间累计净额 + 净流入天数占比
  - 近 7 个自然日区间：{今日-7} ~ {今日}     → 求区间累计净额 + 逐日方向
口径铁律：认准"主力/特大单净主动买入额"，不是毛流入、不是成交额、不是涨幅。
```

**判定矩阵（30 日方向 × 7 日方向）**

| 近30日主力资金 | 近7日主力资金 | 量能配合 | 潜伏含义 | 处置 |
|---|---|---|---|---|
| 持续净流出 | 仍净流出 | 放量下跌 | **持续派发/出货中** | ❌ 一票否决，坚决不埋 |
| 持续净流出 | **转净流入/流出显著放缓** | **缩量企稳** | **流出末端、抛压枯竭、资金回流苗头** | ✅ 潜伏理想信号（左侧经典） |
| 温和净流入 | 延续温和净流入 | 缩量横盘不拉高 | **主力悄悄吸筹中** | ✅ 潜伏最佳，A-2 满分 |
| 温和净流入 | 转净流出 | 放量滞涨 | **吸筹结束转派发 / 阶段见顶** | ⚠️ 谨慎，多半否决 |
| 大幅净流入 | 大幅净流入 | 放量拉升 | **已启动（右侧）** | ➡️ 交《短线严格筛选策略》追涨，不属潜伏 |

**降风险的核心用法**
- **持续净流出 + 仍在放量流出 → 无条件否决**：这是潜伏降风险最重要的一条，避免埋在派发票上被阴跌消耗。
- **"流出末端 + 缩量 + 7 日边际回流"是潜伏的黄金信号**：主力吸筹往往发生在"没人要"的流出末端，等到资金明显流入时价格已抬高。
- **净流入必须"温和"且股价不涨**才算吸筹；若净流入伴随放量拉升，已进入右侧，不再是潜伏。

### 吸筹 vs 出货的关键辨别（必做，防埋错方向）
- ✅ **吸筹特征**：缩量横盘、跌时有承接、股东户数降、跌不动、重心缓慢抬高、30日流出但7日边际回流。
- ❌ **出货特征**：放量滞涨或放量阴跌、股东户数增、反弹无量、重心逐步下移、30日+7日双窗口持续净流出 → **立即否决**。
- ⚠️ **口径校验**：认准"**净主动买入额 / 主力净流入**"，不是毛流入、不是成交额、不是涨幅；区间净额用明确起止日期取。
- 🚨 **警惕"缩量出货"陷阱（`主力资金流分析模块.md` 第六节 T-5）**：**高位**缩量小跌可能是主力"温水煮青蛙"式隐蔽出货，与潜伏要的**低位**缩量（抛压枯竭）形似神异——**位置决定生死**，这正是 P-1"必须下 1/3 分位"的意义。缩量 + 户数增加 + 主力持续净流出 = 出货，不是吸筹。

> 判定规则：**A-1（筹码集中）或 A-2（资金趋势非"持续派发"且出现吸筹/回流迹象）至少一项明确成立为硬门槛**；A-2 命中"持续派发"矩阵格 → **一票否决**（优先级高于其他加分项）；A-3 缩量健康是必要前提（放量下跌一票否决）；A-4 加分。无任何吸筹迹象 → 淘汰（那是死票，不是潜伏位）。

## 闸门 3 — 位置形态闸（低位 + 止跌 + 缩量横盘 + 未启动）

> 潜伏对位置的要求比追涨**更苛刻**——必须是"没启动、位置低、跌无可跌"的埋伏位，赚的就是启动前的低成本。

### 3.1 潜伏期经典 K 线形态（筑底吸筹阶段·用于建底仓）

> 这些形态的共同特征是**在低位、尚未放量启动**，代表抛压枯竭 / 主力吸筹中。命中 ≥1 种，且必须叠加闸门2吸筹迹象 + 下方 P-1~P-4 位置安全。
> 📌 **取数**：`get_stock_performance`（技术形态定性字段：突破反转/连续涨跌/创新低等）+ 日线 OHLC/均线（`stock_highfreq_quotes` 或行情序列）组合识别；`老鸭头/圆弧底/均线粘合`这类复合形态 iFinD 无现成标签，需用「均线关系 + 量能 + K线结构」按下表规则自行判定或人工看图确认。

| 编号 | 形态 | 可勾选特征（识别规则） | 潜伏含义 |
|---|---|---|---|
| L-1 | **低位缩量横盘平台（箱体）** | 长期下跌后进入**缩量箱体横盘 ≥N 日**（如≥15日），振幅收窄、抛压枯竭 | 最典型潜伏形态，平台下沿即止损位 |
| L-2 | **止跌企稳 + 底部抬高（阶梯底）** | 跌势结束，低点逐步抬高、站稳 5/10 日线但**尚未放量突破** | "启动前夜"，赔率最好 |
| L-3 | **超跌缩量筑底（单针探底/双底雏形）** | 深跌后极度缩量、跌无可跌，长下影单针探底或 W 底左脚 | 需配合吸筹迹象，否则是价值陷阱 |
| L-4 | **回踩年线/关键支撑缩量企稳** | 回踩长期均线（年线/半年线）缩量止跌 | 位置低、支撑强、下行空间小 |
| L-5 | **圆弧底（锅底）** | 长期缓慢筑底，K线重心先降后平再缓升，量能"地量→温和放大"，弧度平滑 | 主力低位悄悄吸筹的经典形态，弧底右侧放量即临近点火 |
| L-6 | **均线粘合（缠绕走平）** | 5/10/20/60 日均线在低位**粘合缠绕、间距收窄走平**，股价贴合均线窄幅波动 | 变盘前兆，粘合越久、发散时爆发力越强；向上发散是点火 |
| L-7 | **W 底 / 头肩底（构筑中）** | 双重底第二脚不破前低、缩量；或头肩底右肩缩量回踩，**颈线未破前** | 底部反转结构成型中，颈线是点火线，破颈线前是潜伏位 |
| L-8 | **老鸭头"鸭脖/鸭鼻孔"阶段** | 首波上涨形成"鸭头顶"后回调，5/10 日线下穿又回升、**60 日线始终向上**，回调缩量企稳（鸭鼻孔） | 老鸭头是天然的"潜伏→点火"形态；鸭脖构筑期是潜伏位，鸭嘴张开是点火（见 3.3） |

> **未启动是硬要求**：以上形态一旦**已放量突破**（平台上沿/颈线/鸭头顶/均线向上发散），就进入右侧，交给《短线严格筛选策略》或本策略的"点火加仓"处理，不在筑底期追。潜伏只埋"启动前"。

### 3.2 形态陷阱（一票否决 · 别把出货/中继当筑底）

> 形态的多空含义**高度依赖位置**——同一形态在低位是机会、在高位是陷阱。以下情形命中即否决：

| 陷阱 | 特征 | 为何否决 |
|---|---|---|
| ❌ 高位红三兵/高位形态 | 已大涨后出现的三连阳、假突破 | 高位拉高出货前兆，不是潜伏 |
| ❌ 下降通道中的反弹 | 均线空头排列，仅短线小反弹、60 日线仍向下 | 下跌中继≠筑底，接刀行为 |
| ❌ 假圆弧底/假 W 底 | 弧底/底部放量下跌、重心持续下移、右脚破前低 | 是下跌，不是筑底 |
| ❌ 放量滞涨的伪突破 | 突破当日放巨量但收盘不涨或长上影 | 出货或诱多，非有效点火 |
| ❌ 老鸭头"假鸭嘴" | 60 日线走平/向下时的所谓老鸭头 | 老鸭头成立**必须 60 日线向上**，否则是假形态 |

> **铁律**：形态识别永远和「位置（P-1 下 1/3 分位）+ 60 日线方向 + 量能」三者一起看。脱离位置谈形态没有意义。

### 3.3 位置安全（全部满足才通过，一票否决）

| 编号 | 标准 | 阈值（可调） | 取数工具 |
|---|---|---|---|
| P-1 | **必须在低位区** | 处于近 1 年价格区间**下 1/3 分位**、或距高点回撤显著；**绝不埋高位横盘** | `get_stock_performance` / `stock_highfreq_quotes` |
| P-2 | **止跌确认** | 已止跌企稳（非下跌途中）；下跌未止、破位新低 → 否决（不接刀） | `get_stock_performance` |
| P-3 | **下行空间小、赔率好** | 距最近强支撑（平台下沿/年线）近，止损空间小；上方到目标压力位空间大 | 综合位置测算 |
| P-4 | **缩量健康 + 60 日线不向下** | 横盘期缩量、无放量下跌；**60 日线走平或向上**（形态成立的均线前提） | `stock_highfreq_quotes` |

> 判定规则：潜伏形态命中 ≥1 种（L-1~L-8）**且** 未命中 3.2 任何形态陷阱 **且** 位置安全 P-1~P-4 **全部满足** → 通过，进入打分卡。

## 潜伏打分卡（最终准入）

> 通过全部闸门后，用打分卡量化排序，只埋伏总分达标的票。

| 维度 | 满分 | 评分要点 |
|---|---|---|
| 催化质量（闸门1） | 30 | 催化明确且时间窗 <4 周 +30；<8 周 +20；有催化但时间窗模糊 +10 |
| 吸筹迹象（闸门2） | 30 | 筹码集中 + 多窗口资金趋势(30日温和流入或流出末端回流) + 缩量托底三者齐 +30；两项 +20；一项 +10；30日+7日双窗口持续放量净流出 → 直接淘汰不计分 |
| 位置赔率（闸门3.3） | 20 | 下 1/3 分位 + 止损空间小 + 上方空间大（盈亏比高）+20；中位 +10 |
| 形态质量（闸门3.1） | 12 | 老鸭头 L-8/均线粘合 L-6/圆弧底 L-5/底部抬高 L-2 +12；平台 L-1/W底头肩底 L-7 +10；超跌筑底 L-3/回踩年线 L-4 +8；命中形态陷阱(3.2) 直接淘汰不计分 |
| 排雷边缘度（闸门4） | 8 | 利空已出清 + 质押<10% + 基本面无恶化 → +8；有单项擦边 → +4 |
| **合计** | **100** | |

**准入线（可调）**
- **总分 ≥80** → 一档潜伏候选（可建底仓）。
- **总分 65-79** → 二档（更轻底仓、更严时间止损）。
- **总分 <65** → 放弃。

> 评分是排序工具，不是买入信号。潜伏尤其强调：**分数高也只建底仓，点火信号出现才是真正加仓时点。**

## 盈亏比校验（潜伏的天然优势，仍需量化）

> 潜伏的核心优势就是**赔率**。低位埋伏，下方就是平台/支撑（止损空间小），上方是启动后的第一段（空间大），盈亏比天然优于追高。

### 计算方法
```
上行空间 = 潜伏成本价 → 启动后最近压力位（前平台上沿 / 前高 / 缺口）
下行空间 = 潜伏成本价 → 平台下沿止损位
盈亏比 = 上行空间 ÷ 下行空间
```

### 铁律
```
IF 盈亏比 < 3:1        # 潜伏因承担"时间成本+潜伏失败"风险，赔率门槛比追涨(2:1)更高
THEN 直接放弃，无论催化多诱人
ELSE 进入潜伏建仓纪律
```

> 潜伏要用更高的赔率（≥3:1）来补偿"点火可能落空、横盘可能破位"的额外风险。

## 机械建仓与撤退纪律（潜伏专属·禁止临时改）

> 📌 仓位/熔断/集中度口径统一以 `strategy/资金与风险管理策略.md` 为准；卖出细则见 `strategy/卖出决策策略.md`。本节为潜伏专属流程。

### 分批建仓（潜伏核心：轻仓试错，点火加仓）
```
第一步（潜伏底仓）：闸门全过 + 打分达标 → 建【计划仓位的 1/3~1/2】底仓试错
第二步（点火加仓）：出现"点火信号"（见下）→ 加至目标仓位
第三步（不点火不加）：只要没点火，就维持底仓等待，绝不因"横太久手痒"提前满仓
```

- **单票仓位上限**：一档 ≤15%、二档 ≤8%（潜伏成功前不确定性高，底仓比追涨更轻）。
- **固定风险法**：`底仓金额 = (账户总额 × 单笔风险容忍度) ÷ 止损幅度`，单笔风险容忍度默认 **1%**（可调）。
- 总仓位受环境闸约束；系统性下跌禁区 → 0 仓。

### 点火信号（潜伏 → 加仓/兑现的触发条件）

> 点火 = 潜伏形态**右侧确认**的一刻。以下经典「点火 K 线形态」出现任一，且伴随**放量**（量 > 近期均量 1.5 倍）即视为点火：

| 点火形态 | 识别特征 | 对应潜伏形态 |
|---|---|---|
| **放量突破平台/箱体上沿** | 缩量横盘后首根带量长阳突破上沿 | L-1 平台 |
| **红三兵（低位）** | 低位连续三根重心抬高的阳线、温和放量、不放巨量 | L-2 / L-3 启动初期 |
| **老鸭头"鸭嘴张开"** | 放量向上突破"鸭头顶"、MACD 金叉、均线多头发散（60 日线向上前提下） | L-8 老鸭头 |
| **均线金叉向上发散** | 粘合均线中 5 上穿 10、10 上穿 20，向上打开 | L-6 均线粘合 |
| **突破颈线** | W 底/头肩底放量有效突破颈线 | L-7 双底/头肩底 |
| **圆弧底右侧放量** | 弧底右侧量能温和放大、重心上移突破弧口 | L-5 圆弧底 |

```
IF 出现以上任一点火形态（放量确认）
   OR 催化落地兑现（业绩超预期/政策订单落地）且股价放量响应
   OR 板块启动、目标个股同步放量领涨
THEN 加仓至目标仓位，转入《短线严格筛选策略》的持有/止盈纪律
ELSE 维持底仓，继续潜伏等待
```

> ⚠️ **点火须防假信号**：突破当日放巨量但收盘滞涨/长上影、或 60 日线仍向下时的"伪点火"（见 3.2 形态陷阱）→ 不加仓，维持底仓观察，避免追在诱多上。

### 撤退纪律（潜伏失败的三种出局，无条件机械执行）
```
1) 平台止损：跌破潜伏平台下沿 / 关键支撑（放量破位）→ 无条件清仓，说明埋错了
2) 时间止损：潜伏超过【最长等待期，默认 15~20 个交易日，可调】仍无点火信号 → 撤退
              （潜伏赚的是效率，横太久=资金机会成本，绝不无限期死等）
3) 逻辑证伪：出现出货特征（放量滞涨/放量阴跌/股东户数激增）或催化落空（预期证伪）→ 立即撤
```

- **止损幅度按平台结构定**，优先用平台下沿；配合个股波动率（`get_risk_indicators` 取波动率/beta）自适应，单笔止损硬上限 **-8%**。
- **潜伏失败是常态，不是意外**：底仓轻、止损明确、时间止损硬执行，就能把单次潜伏失败的成本控制到极小，靠"点火成功的大赔率"覆盖多次小止损。

### A股制度性风险应对（同短线，潜伏尤其注意）
- **T+1 + 缩量横盘的流动性风险**：潜伏票往往冷门、成交清淡，破位时可能因流动性差卖不掉。→ 优先选**日均成交额不过分萎缩**的标的，极度缩量到无法正常成交的死票不埋。
- **一字跌停穿损**：对易连续跌停品种（ST/*ST、高质押暴雷）直接在排雷闸规避，不潜伏。

## 潜伏专属风控细则（补齐左侧交易的特有风险）

> 📌 通用口径（固定风险法、三层熔断、集中度、回撤线）仍以 `strategy/资金与风险管理策略.md` 为准。本节只处理**潜伏（左侧+分批+等点火）独有、通用框架没覆盖的风险点**。这几条不落地，实盘会吃暗亏。

### 1. 加仓后止损必须重算（最关键·别把省下的风险吐回去）

> 底仓在低位、止损设平台下沿很合理；但点火加仓时股价已涨上来，若加仓部分仍沿用**原平台下沿**做止损，加仓那笔的实际止损幅度可能被拉到 -15% 以上，一旦是假突破，加仓的钱会大亏，把轻仓省下的优势全吐回去。

```
点火加仓后，立即按"加权平均成本"重算止损，取以下两者中【更高】者为新止损位：
  a) 原平台下沿（结构止损）
  b) 加仓成本 × (1 - 硬上限8%)  → 即整体持仓浮亏不超 -8% 的价位

并复核：加仓后【整体持仓】的单笔风险敞口 ≤ 账户 1%（见第3条），超了就减仓到达标为止。
```
- **点火失败特判**：若加仓后 **3 个交易日内**未能站稳突破位、或跌回平台内（假突破坐实）→ **先减掉加仓部分**回到底仓，不等触整体止损，避免"点火诱多"把底仓也拖亏。

### 2. 连亏熔断改按"金额/回撤"计，不按"笔数"计（消除与通用熔断的冲突）

> 潜伏的盈利结构是"**多次小额止损 + 少数大盈利**"，同时埋 3 只票时，2 周内很容易连吃 3 笔小止损（每笔可能才 -1%~-3%），若直接套用通用的"连亏 3 笔 → 冷静期空仓"，很可能刚好把你锁死在下一只票点火之前。

```
潜伏交易的熔断，用【累计净亏损 / 账户回撤】替代【亏损笔数】：
  - 潜伏组合累计净回撤达账户 -3%   → 暂停新开潜伏仓，存量按纪律处理（对齐通用单日线口径）
  - 潜伏组合累计净回撤达账户 -5%   → 冷静期，全面停开潜伏仓 1~3 日，复盘催化/吸筹判断哪错了
  - 单纯"时间止损"的小额平退（近乎不亏或微亏）不计入"连亏笔数"，只计入金额口径
```
> 与通用文档不冲突：账户级"单日 -3% / 月度 -10%"仍全局生效；此处仅把"连亏 3 笔"这一条对潜伏改成**金额口径**，因为按笔数对高频小止损的潜伏不公平。

### 3. 单笔风险敞口按"目标满仓状态"统一口径（消除底仓/目标仓的歧义）

> 固定风险法的"1% 单笔风险"到底对应底仓还是目标仓位？统一约定如下，避免仓位算错：

```
目标仓位金额 = (账户总额 × 1%) ÷ 止损幅度        # 用【加仓后重算的止损幅度】和目标仓位算
实际执行     = min(目标仓位金额, 单票上限 一档15%/二档8%)   # 与共享文档一致，取小值
底仓         = 目标仓位 × (1/3 ~ 1/2)             # 底仓阶段实际敞口 < 1%，是"预留加仓空间"
```
- 含义：**潜伏的 1% 敞口是以"点火加仓到目标仓位后"的状态为准**；底仓阶段敞口更低，是刻意留出的试错缓冲，不是把 1% 一次性打满。

### 4. 流动性门槛（潜伏票冷门，撤退时才发现卖不掉）

> 潜伏标的常年冷门缩量，等到时间止损/破位要撤时，可能因成交清淡挂单成交不了，纸面止损失效。

```
建仓前硬门槛（可调）：
  - 近 20 日【日均成交额】≥ 5000 万（小盘题材可放宽，但不埋成交额持续萎缩到"无量死票"的标的）
  - 底仓金额 ≤ 该股近 20 日日均成交额的 1%   → 保证自己能顺利进出，不做"自己就是主力"的接盘
超期/破位撤退时：若遇缩量卖不掉 → 分笔挂卖、不砸盘；已破位的用次日集合竞价优先出。
```

### 5. 期望值锚点（给 3:1 赔率一个数学依据）

> "潜伏失败是常态"要量化，否则 3:1 门槛没依据。用期望值反推可接受的点火成功率：

```
期望值 = 点火成功率 × 平均盈利 − 失败率 × 平均亏损
按 赔率 3:1、平均止损 -6% 估：平均盈利≈+18%
  - 点火成功率 30% → 期望 = 0.3×18% − 0.7×6% = +1.2%  → 勉强正期望
  - 点火成功率 40% → 期望 = 0.4×18% − 0.6×6% = +3.6%  → 健康
结论：潜伏的点火成功率经验目标 ≥35%（可调），低于此则须提高赔率门槛或收紧催化闸。
```
> ⚠️ 上述为经验测算，非严谨统计。**每季度用复盘记录的真实点火成功率回填**（见文末季度校准），让门槛个人化。

### 6. 潜伏底仓池总额度上限（防"多个底仓悄悄堆高总仓"）

> 底仓都趴着不动、各自占额度，多个潜伏叠加会让总仓位在不知不觉中升高，且往往押注同一逻辑（都赌某景气反转），催化落空时集体阴跌。

- **潜伏底仓合计额度 ≤ 账户 30%**（可调）；点火成功转入持有的仓位不占此额度（已转为常规持仓）。
- **同一催化逻辑/同一产业链的潜伏合计 ≤ 账户 15%**——三只票赌同一件事，本质是一笔集中下注，不是分散。
- 跨线/同逻辑总敞口仍受《策略路由》"跨线集中度 ≤35%"约束（含潜伏底仓 + 已点火持仓 + 中长线同赛道持仓）。

## 潜伏 vs 追涨（一表看清定位）

| 维度 | 短线潜伏策略（本文·左侧埋伏） | 短线严格筛选策略（右侧突破） |
|---|---|---|
| 买入时机 | 启动**前**（缩量横盘吸筹期） | 启动**时**（放量突破扳机） |
| 位置 | 低位、下 1/3 分位 | 突破位、可放宽至启动涨幅 <50% |
| 环境偏好 | 震荡/退潮/冰点期反而好 | 进攻期、赚钱效应好 |
| 核心闸 | 吸筹闸（有人悄悄买） | 资金闸（爆发式净流入）+ 形态扳机 |
| 赔率门槛 | ≥3:1（补偿潜伏失败风险） | ≥2:1 |
| 建仓 | 分批：底仓试错 → 点火加仓 | 扳机触发一次性买入 |
| 止损 | 平台下沿 + **时间止损**（横太久撤） | 破买入逻辑 / 机械 -5~-8% |
| 主要风险 | 潜伏失败、横盘破位、催化落空 | 追高、突破失败被套 |
| 收益来源 | 从潜伏到点火的**第一段** | 突破后的主升段 |

## 一页速查（口袋版）

```
环境闸：系统性下跌(破60日线加速)禁区；震荡/退潮/冰点反而是潜伏好窗口
赛道闸：必须有"明确+临近(<8周)的催化预期"，无催化=价值陷阱，不埋
排雷闸：减持/质押>40%/处罚/财务恶化/利空未出清 → 命中即出局（先跑，比追涨更严）
吸筹闸：股东户数降/机构增 + 多窗口资金趋势(30日+7日) + 缩量托底
       主力资金：30日+7日双窗口持续净流出=派发→否决；流出末端7日回流/温和流入不拉高=吸筹✓
形态闸：潜伏形态(平台/圆弧底/均线粘合/W底头肩底/老鸭头鸭脖/底部抬高)任1，未启动
位置闸：下1/3分位 + 止跌企稳 + 60日线不向下 + 缩量；高位红三兵/下降通道反弹/假突破 → 否决
点火形态：放量破平台/低位红三兵/老鸭头鸭嘴张开/均线金叉发散/破颈线 → 才加仓
盈亏比：≥3:1 才做（比追涨2:1更高，补偿潜伏失败）
打分卡：≥80一档建底仓，65-79二档，<65放弃
建仓：底仓1/3~1/2试错 → 点火信号出现才加仓；1%敞口按目标满仓算
加仓止损：点火加仓后按加权成本重算止损(取平台下沿与成本-8%的更高者)；3日不站稳先减加仓部分
撤退：破平台/横盘超15~20日无点火/出现出货特征 → 无条件撤
熔断：潜伏组合累计回撤-3%停开新仓、-5%冷静期(按金额不按笔数)
额度：潜伏底仓合计≤30%、同一逻辑≤15%；流动性门槛日均成交额≥5000万
```

> **找催化、验吸筹、埋低位、轻底仓、等点火、守时限**——潜伏的纪律核心是：**用轻仓和硬时间止损把"埋错"的成本压到最小，靠"点火成功的大赔率"赚钱。宁可撤 100 次，不死等 1 次。**
