# 策略路由（当前适合做什么）

> 生成日期：2026-07-01
> 在动手选股之前，先用本文判断"当前市场环境适合用哪套策略"。避免逆风硬做。
> **⚠️ 重要声明：本文不构成任何投资建议。市场环境判断存在滞后性，请独立决策、盈亏自负。**

---

## 核心原则

> **环境决定策略，策略决定动作。** 不是"我想做短线"就做短线，而是"当前环境允许做短线"才做。

> 📌 **环境判断口径统一引用 `strategy/大盘环境模块.md`**：先算「环境分」定分级，再按下方规则路由到对应策略。三条线的环境闸与本文的环境判断共用同一份事实与口径。

---

## 路由决策表

| 市场环境特征 | 适合的策略 | 不适合的策略 |
|-------------|-----------|-------------|
| 大盘强势 + 有持续主线 + 赚钱效应好 | **短线追涨**（资金主线跟踪·右侧突破） | — |
| 大盘震荡/退潮/冰点 + 有明确临近催化 + 低位吸筹迹象 | **短线潜伏**（左侧埋伏·等点火） | 短线右侧追高 |
| 行业景气向上 + 个股趋势确立 + 估值不透支 | **中线**（趋势+景气跟随） | 短线追高位妖股 |
| 优质公司估值低分位 + 基本面健康 | **长线**（价值成长建仓） | 短线频繁交易 |
| 大盘弱势 + 无主线 + 缩量阴跌 | **风险冻结**：禁止短线追涨/潜伏、中线和长线新开仓或加仓；存量仓位按风险总纲处理 | 新增任何风险敞口 |
| 结构市（部分板块强、整体弱） | **中线**仅做最强景气主线 | 短线全面出击 |

## 快速判定流程

```text
第 1 步：大盘环境
├── 指数站上 20/60 日线 + 量能正常 → 可以做
├── 指数跌破 20 日线加速下行 → 谨慎/降仓
└── 指数跌破 60 日线 + 缩量 → 空仓等待

第 2 步：最新金融新闻与情绪修正
├── 记录 decision_time，按策略窗口调用 search_news
├── 去重后识别 L1/L2 风险、受益方向、已兑现与消息/盘面背离
├── 重大公司事件再用 search_notice / get_stock_events 核验
└── 输出“偏积极/中性/偏谨慎/高风险”；失败或时间不明时显式降级

第 3 步：有没有主线
├── plate-rotation 双源共振存在持续主线 → 短线/中线窗口开
└── 无持续主线、板块轮动混乱 → 短线窗口关

第 4 步：行业景气
├── 有景气向上 + 业绩拐点的行业 → 中线窗口开
└── 景气普遍见顶 → 中线窗口关

第 5 步：估值环境
├── 有 PEG<1 且基本面好的公司 → 长线窗口开
└── 普遍高估 → 长线只观察不买
```

> 新闻情绪不是第五个环境分值维度，只用于分级降档、仓位收缩、风险禁区和方向观察。完整取数、去重与输出契约见
> `strategy/消息面事件驱动模块.md`。

## 路由 if-then 规则

```
IF 环境分 <30
THEN → 进入全局风险冻结：短线追涨/潜伏、中线、长线均禁止新开仓和加仓；存量仓位只按《资金与风险管理策略》执行持有、减仓或卖出

IF 环境分 ≥30 AND 环境闸通过（大盘不弱）AND 有双源共振主线 AND 赚钱效应好
THEN → 启用「短线严格筛选策略」（右侧追涨）

IF 大盘未系统性破位（震荡/退潮/冰点均可）AND 目标有明确临近催化（<8周）AND 出现低位吸筹迹象
THEN → 启用「短线潜伏策略」（左侧埋伏，轻底仓等点火）

IF 行业景气向上 AND 个股趋势确立（均线多头）AND 业绩有拐点
THEN → 启用「中线严格筛选策略」

IF 优质公司（ROE>15%+护城河）AND 估值合理（PEG<1 或低分位）
THEN → 启用「长线严格筛选策略」

IF 拿到一只具体的票想全面了解
THEN → 启用「个股综合体检策略」

IF 以上条件都不满足
THEN → 空仓等待，不强行操作
```

## 多策略并行规则

- 可以**同时**持有短线仓 + 中线仓 + 长线仓，但每个仓位必须按**各自的纪律**执行
- **绝不混用**：短线票破位不能改口"做中线"；中线票破位不能改口"做长线"
- 总仓位上限仍受环境约束：弱势环境即使各线都有候选，也要控制总仓位

### 跨线集中度约束（防"假分散"）

> 三条线分别选股，容易在不知不觉中押注**同一条产业链**（如短线买算力龙头、中线买算力景气股、长线买 AI 核心资产），看着分散、实则集中，系统性回调时一起爆。

- **单一行业/产业链的跨线合计敞口 ≤35%**（含短+中+长三线在该赛道的全部持仓）。
- 建仓前先检查：新标的所属行业，与现有持仓（跨全部线）是否高度重叠 / 强相关。
- 若某赛道跨线敞口已达上限 → 新候选即使单独打分达标，也**只进观察池**，不再加仓该赛道。
- 相关性判断优先用行业分类（`get_stock_info` 所属行业 + `sector_data` 板块归属）辅助。

## 一句话总览

> **大盘强+有主线 → 短线追涨；震荡冰点+有催化+低位吸筹 → 短线潜伏；景气+趋势 → 中线；便宜+好公司 → 长线；都不满足 → 空仓。** 先判环境再选策略，不逆风硬做。

> 📌 **短线两条腿**：进攻期主线启动了用《短线严格筛选策略》右侧追；市场冷清、好票还趴着时用《短线潜伏策略》左侧埋。两者共用排雷/资金/风控口径，区别只在"买在启动前还是启动时"。
