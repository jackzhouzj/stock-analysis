# 策略路由（当前适合做什么）

> 生成日期：2026-07-01
> 在动手选股之前，先用本文判断"当前市场环境适合用哪套策略"。避免逆风硬做。
> **⚠️ 重要声明：本文不构成任何投资建议。市场环境判断存在滞后性，请独立决策、盈亏自负。**

---

## 核心原则

> **环境决定策略，策略决定动作。** 不是"我想做短线"就做短线，而是"当前环境允许做短线"才做。

> 📌 **环境判断口径统一引用 `strategy/大盘环境模块.md`**：先算 A 股四维「环境分」，再依次经过 `strategy/外围市场影响模块.md` 的价格风险覆盖和 `strategy/消息面事件驱动模块.md` 的事件修正，最后路由到对应策略。外围与消息均不改变原始分值。

> 📌 用户明确询问“明后天、未来一两天或未来1—3个交易日走势”时，优先进入 `strategy/未来1到3个交易日行情研判策略.md`。该策略可以在任何环境下描述风险情景，但研判结论不是开仓许可；需要交易动作时仍须回到本路由和对应原子策略。

## 路由决策表

| 市场环境特征 | 适合的策略 | 不适合的策略 |
|-------------|-----------|-------------|
| 大盘强势 + 有持续主线 + 赚钱效应好 + 盘中结构确认 | **超短线**（09:31—09:40开盘确认 / 14:31—14:50尾盘确认，持有1—3日） | 无候选池临时追涨、09:30整抢单 |
| 大盘强势 + 有持续主线 + 赚钱效应好 | **短线追涨**（资金主线跟踪·右侧突破） | — |
| 大盘震荡/退潮/冰点 + 有明确临近催化 + 低位吸筹迹象 | **短线潜伏**（左侧埋伏·等点火） | 短线右侧追高 |
| 行业景气向上 + 个股趋势确立 + 估值不透支 | **中线**（趋势+景气跟随） | 短线追高位妖股 |
| 优质公司估值低分位 + 基本面健康 | **长线**（价值成长建仓） | 短线频繁交易 |
| 需要长期beta或跨资产分散，且产品质量/趋势通过 | **基金ETF**（核心—卫星—防守轮动） | 同指数重复配置、追单日大涨 |
| 大盘弱势 + 无主线 + 缩量阴跌 | **风险冻结**：禁止短线追涨/潜伏、中线和长线新开仓或加仓；存量仓位按风险总纲处理 | 新增任何风险敞口 |
| 结构市（部分板块强、整体弱） | **中线**仅做最强景气主线 | 短线全面出击 |

## 快速判定流程

```text
第 1 步：大盘环境
├── 指数站上 20/60 日线 + 量能正常 → 可以做
├── 指数跌破 20 日线加速下行 → 谨慎/降仓
└── 指数跌破 60 日线 + 缩量 → 空仓等待

第 2 步：外围市场价格风险覆盖
├── 按各市场独立交易日历读取港股、美股、韩国等已完成数据
├── 输出O0—O4、A/B/C置信度、行业禁区和有效期
├── O1只给观察方向；O3/O4只在A/B级时执行风险动作
└── 韩国或海外实时能力缺失时明确降级，不把无数据当平盘

第 3 步：最新金融新闻与情绪修正
├── 记录 decision_time，按策略窗口调用 search_news
├── 解释外围价格原因、识别L1/L2风险与兑现状态
├── 重大公司事件再用 search_notice / get_stock_events 核验
└── 通过overlay_id合并同一冲击，不重复下调

第 4 步：有没有主线
├── plate-rotation 双源共振存在持续主线 → 超短线/短线/中线窗口开
├── 位于09:31—09:40或14:31—14:50且分时确认完整 → 可路由超短线
└── 无持续主线、板块轮动混乱 → 超短线与短线窗口关

第 5 步：行业景气
├── 有景气向上 + 业绩拐点的行业 → 中线窗口开
└── 景气普遍见顶 → 中线窗口关

第 6 步：估值环境
├── 有 PEG<1 且基本面好的公司 → 长线窗口开
└── 普遍高估 → 长线只观察不买

第 7 步：基金角色
├── 需要宽基beta且环境未冻结 → 基金核心仓分批
├── 卫星ETF趋势和相对强度通过 → 小仓轮动
└── 环境偏弱或权益高度重叠 → 国债/货币ETF防守
```

> 外围价格与新闻情绪都不是环境分值维度。外围层负责价格风险覆盖，新闻层负责事件解释和二次修正；二者只允许执行一次最终风险动作。完整契约见 `strategy/外围市场影响模块.md` 与 `strategy/消息面事件驱动模块.md`。

## 路由 if-then 规则

```
IF 用户明确询问未来1—3个交易日的大盘走势、强势行业板块或板块龙头
THEN → 优先启用「未来1到3个交易日行情研判策略」；按“大盘H1—H3 → 双源板块排序 → 龙头分类复核”输出。若继续询问买卖动作，再进入对应原子策略，不把研判等同开仓信号

IF 环境分 <30
THEN → 进入全局风险冻结：超短线、短线追涨/潜伏、中线、长线均禁止新开仓和加仓；存量仓位只按《资金与风险管理策略》执行持有、减仓或卖出

IF 外围覆盖=O4 AND 置信度∈{A,B}
THEN → 超短线/短线冻结新仓；中线、长线和权益卫星仓暂停加仓；存量不机械清仓

IF 外围覆盖=O3 AND 置信度∈{A,B}
THEN → 新仓风险预算按台账折扣，受冲击行业进入临时禁区；与A股原路由取更严格动作

IF 外围覆盖=O0 OR 置信度=C
THEN → 不得判定中性；依调用策略的数据完整性要求降为观察或保守额度

IF 环境分 ≥75 AND 有双源共振持续主线 AND 赚钱效应好 AND 时间位于09:31—09:40或14:31—14:50
THEN → 启用「超短线选股策略」；T-1候选池先过硬闸，再按M开盘确认或A尾盘确认执行，计划持有1—3日

IF 环境分 50—74 AND 仅有局部最强主线
THEN → 超短线只允许防守额度的一档核心；无完整盘中确认则不开仓

IF 环境分 ≥30 AND 环境闸通过（大盘不弱）AND 有双源共振主线 AND 赚钱效应好
THEN → 启用「短线严格筛选策略」（右侧追涨）

IF 大盘未系统性破位（震荡/退潮/冰点均可）AND 目标有明确临近催化（<8周）AND 出现低位吸筹迹象
THEN → 启用「短线潜伏策略」（左侧埋伏，轻底仓等点火）

IF 行业景气向上 AND 个股趋势确立（均线多头）AND 业绩有拐点
THEN → 启用「中线严格筛选策略」

IF 优质公司（ROE>15%+护城河）AND 估值合理（PEG<1 或低分位）
THEN → 启用「长线严格筛选策略」

IF 用户要选基金、分析ETF或执行基金轮动
THEN → 启用「基金严格筛选策略」；核心仓看产品质量与配置需求，卫星仓必须再过趋势和相对强度闸

IF 拿到一只具体的票想全面了解
THEN → 启用「个股综合体检策略」

IF 以上条件都不满足
THEN → 空仓等待，不强行操作
```

## 多策略并行规则

- 可以**同时**持有超短线仓 + 短线仓 + 中线仓 + 长线仓，但每个仓位必须按**各自的纪律**执行
- **绝不混用**：超短线到期不能改口“普通短线”；短线票破位不能改口“做中线”；中线票破位不能改口“做长线”
- 总仓位上限仍受环境约束：弱势环境即使各线都有候选，也要控制总仓位

### 跨线集中度约束（防"假分散"）

> 多条线分别选股，容易在不知不觉中押注**同一条产业链**（如超短线和短线买算力龙头、中线买算力景气股、长线买 AI 核心资产），看着分散、实则集中，系统性回调时一起爆。

- **单一行业/产业链的跨线合计敞口 ≤35%**（含超短线、短线、中线、长线及基金卫星仓在该赛道的全部持仓）。
- 建仓前先检查：新标的所属行业，与现有持仓（跨全部线）是否高度重叠 / 强相关。
- 若某赛道跨线敞口已达上限 → 新候选即使单独打分达标，也**只进观察池**，不再加仓该赛道。
- 相关性判断优先用行业分类（`get_stock_info` 所属行业 + `sector_data` 板块归属）辅助。

## 一句话总览

> **用户问未来1—3日时，先做大盘H1—H3情景，再找双源共振且有持续性的行业板块，最后按趋势/容量/弹性三类复核龙头；用户问交易动作时，才继续按环境选择超短线、短线、中线或风险冻结。**

> 📌 **短周期三种模式**：09:31/14:31双时点确认用《超短线选股策略》；进攻期主线启动用《短线严格筛选策略》右侧追；市场冷清、好票未启动时用《短线潜伏策略》左侧埋。三者共用排雷/资金/风控口径，但决策时点、持有周期和阈值不得混用。
