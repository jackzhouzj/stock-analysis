# 策略路由（当前适合做什么）

> 生成日期：2026-07-01
> 在动手选股之前，先用本文判断"当前市场环境适合用哪套策略"。避免逆风硬做。
> **⚠️ 重要声明：本文不构成任何投资建议。市场环境判断存在滞后性，请独立决策、盈亏自负。**

---

## 核心原则

> **环境决定策略，策略决定动作。** 不是"我想做短线"就做短线，而是"当前环境允许做短线"才做。

---

## 路由决策表

| 市场环境特征 | 适合的策略 | 不适合的策略 |
|-------------|-----------|-------------|
| 大盘强势 + 有持续主线 + 赚钱效应好 | **短线**（资金主线跟踪） | — |
| 行业景气向上 + 个股趋势确立 + 估值不透支 | **中线**（趋势+景气跟随） | 短线追高位妖股 |
| 优质公司估值低分位 + 基本面健康 | **长线**（价值成长建仓） | 短线频繁交易 |
| 大盘弱势 + 无主线 + 缩量阴跌 | **空仓等待** | 任何策略都不做 |
| 结构市（部分板块强、整体弱） | **中线**仅做最强景气主线 | 短线全面出击 |

---

## 快速判定流程

```
第 1 步：大盘环境
├── 指数站上 20/60 日线 + 量能正常 → 可以做
├── 指数跌破 20 日线加速下行 → 谨慎/降仓
└── 指数跌破 60 日线 + 缩量 → 空仓等待

第 2 步：有没有主线
├── plate-rotation 双源共振存在持续主线 → 短线/中线窗口开
└── 无持续主线、板块轮动混乱 → 短线窗口关

第 3 步：行业景气
├── 有景气向上 + 业绩拐点的行业 → 中线窗口开
└── 景气普遍见顶 → 中线窗口关

第 4 步：估值环境
├── 有 PEG<1 且基本面好的公司 → 长线窗口开
└── 普遍高估 → 长线只观察不买
```

---

## 路由 if-then 规则

```
IF 环境闸通过（大盘不弱）AND 有双源共振主线 AND 赚钱效应好
THEN → 启用「短线严格筛选策略」

IF 行业景气向上 AND 个股趋势确立（均线多头）AND 业绩有拐点
THEN → 启用「中线严格筛选策略」

IF 优质公司（ROE>15%+护城河）AND 估值合理（PEG<1 或低分位）
THEN → 启用「长线严格筛选策略」

IF 拿到一只具体的票想全面了解
THEN → 启用「个股综合体检策略」

IF 以上条件都不满足
THEN → 空仓等待，不强行操作
```

---

## 多策略并行规则

- 可以**同时**持有短线仓 + 中线仓 + 长线仓，但每个仓位必须按**各自的纪律**执行
- **绝不混用**：短线票破位不能改口"做中线"；中线票破位不能改口"做长线"
- 总仓位上限仍受环境约束：弱势环境即使各线都有候选，也要控制总仓位

## 取数工具映射

| 判断维度 | 工具 | 用途 |
|----------|------|------|
| 大盘趋势 | `index_data`（沪深300/创业板指） | 判断指数位置与均线关系 |
| 量能 | `index_highfreq_quotes` | 判断缩量/放量 |
| 板块主线 | `plate-rotation` `today`/`curve` | 判断是否有持续主线 |
| 行业景气 | `get_edb_data` / `sector_data` | 判断中线景气窗口 |
| 估值环境 | `get_stock_financials` | 判断长线建仓窗口 |

---

## 一句话总览

> **大盘强+有主线 → 短线；景气+趋势 → 中线；便宜+好公司 → 长线；都不满足 → 空仓。** 先判环境再选策略，不逆风硬做。
