# 股票常用指标手册（怎么看 + 怎么用来选股）

> 生成日期：2026-06-23
> 配套《短线炒股攻略》《长线投资攻略》使用：本手册讲"指标怎么读、阈值怎么定、组合怎么选股"。
> **⚠️ 重要声明：本文为指标科普与方法论，不构成任何投资建议、不推荐个股。指标是参考工具非买卖信号，阈值随行业/周期变化。投资有风险，盈亏自负。**

---

## 一、估值指标（贵不贵）

| 指标 | 含义 | 怎么看 | 注意 |
|---|---|---|---|
| **PE（市盈率）** | 股价/每股收益 | 越低越便宜；看 TTM 滚动 + 历史分位数 | 周期股低PE常是顶部；亏损股PE无意义 |
| **PB（市净率）** | 股价/每股净资产 | 重资产/金融股常用；<1 破净 | 轻资产公司PB高属正常 |
| **PS（市销率）** | 股价/每股营收 | 未盈利成长股（如创新药）用 | 不反映盈利能力 |
| **PEG** | PE / 净利增速 | **<1 偏低估，是成长股黄金指标** | 增速须真实可持续 |
| **股息率** | 每股分红/股价 | 高股息=防御/价值，>3%较优 | 看分红是否可持续 |

> **关键用法**：单看 PE 没意义，要看**历史分位数**（当前 PE 在过去 5 年的百分位）+ **行业横向对比** + **PEG 配合增速**。

## 二、盈利能力指标（赚钱能力强不强）

| 指标 | 含义 | 怎么看 | 优秀标准 |
|---|---|---|---|
| **ROE（净资产收益率）** | 净利润/净资产 | **最核心质量指标**，长期>15%且稳定 | >15% 优秀，>20% 卓越 |
| **ROA** | 净利润/总资产 | 资产使用效率 | 越高越好 |
| **毛利率** | (营收-成本)/营收 | 护城河/定价权的投影，高且稳=有壁垒 | 因行业而异，看趋势 |
| **净利率** | 净利润/营收 | 综合盈利能力 | 稳定提升为佳 |
| **ROIC** | 投入资本回报率 | 比 ROE 更严谨（剔除杠杆） | >WACC 才创造价值 |

> **杜邦分析**：ROE = 净利率 × 总资产周转率 × 权益乘数（杠杆）。看 ROE 高是靠**真盈利**还是靠**高杠杆堆出来的**——后者风险大。

## 三、成长性指标（长不长得动）

| 指标 | 怎么看 | 陷阱提示 |
|---|---|---|
| **营收同比增长** | 真成长的根基，应持续>20% | 营收降而净利增=不可持续 |
| **净利润同比增长** | 越高越好 | ⚠️**先看营收增速辨真假**，净利暴增多为低基数/扭亏/非经常损益 |
| **净利润增速逐季加速** | 戴维斯双击的发动机 | 加速比单纯高增更值钱 |
| **扣非净利润** | 剔除非经常损益的真实盈利 | 扣非<净利润太多=靠卖资产/补贴 |

> **辨别假成长**：净利+2000% 但营收只+50%、且去年同期接近亏损 → 低基数陷阱，不是真高成长。

## 四、财务健康指标（会不会暴雷）

| 指标 | 怎么看 | 警戒线 |
|---|---|---|
| **资产负债率** | 负债/总资产 | 过高有偿债风险（除金融/地产） |
| **经营现金流/净利润** | 利润含金量 | **<0.8 警惕**，利润是应收账款堆的 |
| **应收账款+存货占比** | 占流动资产比重 | 畸高=营运资本沉淀+减值风险 |
| **商誉/净资产** | 并购溢价 | 过高有减值暴雷风险 |
| **流动/速动比率** | 短期偿债能力 | 流动比率<1 注意流动性 |

## 五、筹码与资金指标（谁在买）

| 指标 | 怎么看 | 信号 |
|---|---|---|
| **主力资金净流入（特大单）** | 看是否**持续**净流入 | 持续流入>脉冲对倒；流出=资金撤离 |
| **股东户数** | 下降=筹码集中 | 户数降+户均持股升=主力吸筹 |
| **机构持仓比例** | 启动前低、之后升=机构认可 | 骤降=机构撤离（利空） |
| **龙虎榜/北向** | 游资席位、外资动向 | 净买入=资金进攻 |
| **质押比例** | 控股股东质押占比 | **>40% 警惕**爆仓风险 |

> ⚠️ 区分口径：板块"强度分/涨幅" ≠ "资金净流入额"；"流入额(合计)"常是毛流入/成交额，要认准"**净主动买入额**"。

## 六、技术指标（什么时候买卖·辅助）

| 指标 | 怎么看 | 用途 |
|---|---|---|
| **均线（MA）** | 多头排列(短>长)看多；回踩支撑 | 趋势判断 |
| **MACD** | 金叉看多/死叉看空；红绿柱看动能 | 趋势+背离 |
| **KDJ/RSI** | 超买(>80)超卖(<20) | 短线择时、背离 |
| **成交量** | **价涨量增健康；放天量滞涨警惕** | 量价配合是核心 |
| **换手率** | 活跃度；高换手+高位=分歧 | 判断资金活跃 |
| **布林带** | 上下轨+中轨 | 波动区间、突破 |

> 技术指标是**辅助择时**，不是选股核心；脱离基本面和资金面单用技术，胜率不稳定。

## 七、如何组合指标选到好股票（实战流程）

### 长线选股（价值成长）
```
1. 行业：高景气、大空间、格局向龙头集中
2. 质量：ROE>15%（长期稳定）+ 毛利率高且稳 + 经营现金流/净利>0.8
3. 成长：营收+利润双高增（营收增速验真假）+ 增速可持续
4. 估值：PEG<1 或 PE 历史低分位
5. 排雷：无减持/质押/造假/商誉雷
6. 买点：估值合理区分批建仓
```

### 短线选股（资金趋势）
```
1. 环境：大盘不弱、有赚钱效应
2. 板块：资金主线（双源共振 + 特大单持续净流入）
3. 个股：形态触发（突破/回踩）+ 主力资金流入 + 位置不高
4. 排雷：无重大利空
5. 纪律：机械止损止盈
```

### 一个通用的"好股票"快速画像
> **高 ROE（>15%）+ 营收利润双增 + 高毛利 + 现金流健康 + 估值 PEG<1 + 无治理雷 + 资金认可** —— 同时满足越多条，是好股票的概率越大。

## 八、指标使用的 6 条铁律

1. **不单看一个指标**：高 ROE 可能靠高杠杆、低 PE 可能是价值陷阱——必须组合验证。
2. **看趋势比看绝对值重要**：ROE/毛利率是在升还是在降，比当前数值更说明问题。
3. **同行业横向比**：跨行业比 PE/毛利率没意义（银行 vs 软件）。
4. **看历史分位数**：估值贵不贵要和它自己的历史比。
5. **财务指标看"扣非"和"现金流"**：剔除一次性损益，确认利润是真金。
6. **指标是过去时**：反映的是已发生的经营，需结合行业趋势与前瞻判断。

---

## 一句话总览

> **估值看 PE/PEG，质量看 ROE/毛利，成长看营收增速，安全看现金流/负债，资金看净流入/筹码**——组合验证、看趋势、同行比、查历史分位，再叠加排雷，才能筛出真正的好股票。
