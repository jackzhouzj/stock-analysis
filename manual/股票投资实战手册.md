## 二、一页速查表（精华浓缩）

### 选股核心指标阈值
| 维度 | 指标 | 好股票参考 |
|---|---|---|
| 盈利质量 | ROE | 长期 >15%、稳定/提升 |
| 盈利质量 | 毛利率 | 高且稳（护城河） |
| 成长 | 营收同比 | >20%，且利润同步增 |
| 成长 | 净利真伪 | **先看营收增速排除低基数** |
| 估值 | PEG | <1（成长股黄金区） |
| 安全 | 经营现金流/净利 | >0.8 |
| 资金 | 主力净流入 | 持续>脉冲，认准"净主动买入" |

### 排雷清单（一票否决）
- 大股东密集减持（尤其"业绩好却猛减"）
- 财务造假 / 信披违规 / 监管处罚
- 控股股东高质押（>40%）
- 净利暴增靠低基数 / 营收反降
- 经营现金流长期背离净利润
- 商誉过高 / 频繁并购买利润

### 短线三共振（缺一不可）
> **形态触发（突破/回踩）+ 主力资金持续净流入 + 所在板块是资金主线**
> 形态是扳机，板块资金是胜率开关，位置决定赔率。

### 长线四问（买前自检）
> 行业空间够大吗？公司护城河在变宽吗？利润是真金（现金流）吗？现在价格便宜吗（PEG/分位）？

## 三、贯穿全书的 5 条核心纪律

1. **数据验证，不靠感觉**：板块、资金、公告，先看事实再下结论。
2. **交叉验证，不信单一指标/单一数据源**：高ROE可能靠杠杆、低PE可能是陷阱、强度≠净流入。
3. **辨别真假成长**：净利暴增先看营收增速，剔除低基数/扭亏。
4. **排雷优先于选股**：避免亏大钱比抓涨更重要。
5. **长短线不混用**：长线票别短炒，短线票套牢别改口"价值投资"。

## 四、本手册的数据与局限说明（诚实披露）

- 指标阈值为经验参考，随行业/周期变化，**非固定标准**。
- 资金流数据存在口径差异（净额 vs 毛流入/成交额），行业级净流入排名接口不稳定，本手册案例采用**自下而上的个股净额**口径。
- 形态"胜率"为经验区间，**非严谨统计**，且强依赖大盘环境。
- 大牛股/上涨**无法被预测**，所有筛选是"符合历史概率特征"，不是"会涨"。
